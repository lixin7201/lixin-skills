# 转折与推进规则

## 常用推进链

1. 现象 -> 扎心处 -> 现实账本。
2. 个人经历 -> 身边人 -> 一类人。
3. 北京案例 -> 全国趋势 -> 普通家庭自保。
4. 热门选择 -> 几年后的反噬 -> 别跟风。
5. 情绪共鸣 -> 冷一点的判断 -> 保守建议。

## 常用转折句

- 但问题在于，事情并没有这么简单。
- 更扎心的是，真正受影响的是普通家庭。
- 这就导致一个很现实的问题。
- 你看，这事最后还是落到了钱和时间上。
- 所以我现在越来越不建议大家上头。
- 当然，这只是我个人经验，不一定适合所有人。

## 推进禁忌

- 不要靠“第一、第二、第三”堆满全文；只有盘点型文章才用。
- 不要用过强阴谋论解释现实。
- 不要从一个个案直接推全国结论，必须加边界。
