# 大宜宾雯雯-DNA

## 证据等级

- 训练样本：401 篇，来自该小编独立 top60 池扣除 holdout。
- 样本门槛：超过 20 篇，可生成稳定小编 DNA。
- 主要类型：综合本地新闻277; 政务公告党建85; 城建交通更新13; 民生服务提醒13; 突发应急安全5。

## 小编定位

- 路线：政务公益型
- 核心写稿方式：不抢情绪，先给来源和核心事实；适合政务党建、公益求助、权威信息转载。
- 擅长场景：大美宜宾、政务公告、公益求助、人民日报/新华社式信息

## 思考框架

1. 先判断这条素材和本地用户的关系，不直接套模板。
2. 找一个最容易接话的入口：经历、选择、问题、利益、风险或城市变化。
3. 事实先落地，情绪后跟；涉及高风险内容只写已确认事实。
4. 结尾必须把评论区打开，但问题要具体。

## 结构打法

来源/当事人身份 -> 事实经过 -> 政策或困难细节 -> 行动渠道。

## 不像的信号

- 写成其他小编的路线。
- 开头没有本地对象或用户关系。
- 评论问题空泛。
- 事实边界不清，尤其是投诉、突发、招聘、公益。
