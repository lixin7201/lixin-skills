# 账号选题判断框架

## 选题优先级

1. 上级关注：新兴领域党建、互联网行业党建、网络统战、妇联/工会/团建联动。
2. 组织动作：主题党日、学习培训、组织生活、党员志愿服务、读书分享。
3. 业务融合：就业服务、婚恋服务、消费赋能、政务传播、直播助农、内容合规。
4. 外部认可：领导调研、荣誉表彰、优秀案例、品牌建设。
5. 内部成长：青年党员成长、员工学习、岗位能力、团队凝聚。

## 选角度的四个问题

- 这件事体现了哪个党建动作？组织建设、思想学习、党员先锋、党群联动、品牌创建。
- 它落到了哪个民生/业务场景？就业、婚恋、消费、公益、传播、合规。
- 谁被服务或被带动？群众、求职者、青年、女性、企业、困难群体、员工。
- 下一步能接什么？持续深化、常态化开展、做优品牌、服务地方发展。

## 不同素材的取舍

- 活动流程完整：按时间线写，最后转党建意义。
- 荣誉素材强：先写荣誉和权威认可，再回看长期积累。
- 领导调研：先写领导身份和调研主题，再写参观/汇报/肯定。
- 读书学习：先写学习主题和参与者，再写个人分享如何回到岗位。
- 发言材料：先写背景和痛点，再写做法、成效、启示。

## 不能选的角度

- 只把党建写成万能装饰词，看不出具体活动。
- 只夸企业厉害，不写组织建设和党员作用。
- 只写领导来过，不写调研主题、汇报内容和下一步承接。
- 只写读了什么书，不写岗位转化。
- 只写“温暖、力量、赋能”，没有对象、动作、结果。
