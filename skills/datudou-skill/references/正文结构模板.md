# 正文结构模板

## 结构指标

来自训练集 `data/结构与段落指标.md`：

| 指标 | 约束 |
|---|---|
| 中位字数 | 约 948 中文字 |
| 段落中位长度 | 约 52 字 |
| 每千字段落数 | 约 17 |
| <=20 字短段比例 | 约 7% |
| 连续短段 | 通常不超过 3 段 |

## 主结构 1：现实账本

```text
入口问题
-> 给公式/分类
-> 解释每类家庭怎么受影响
-> 举一个身边例子
-> 说明这个判断的局限
-> 评论区落点
```

## 主结构 2：趋势拆解

```text
最近事件
-> 这不是孤例
-> 变量一：人口/供需/政策
-> 变量二：钱/资产/收入
-> 变量三：代际/家庭责任
-> 普通人该怎么保守处理
```

## 主结构 3：个人随笔

```text
个人小事
-> 自嘲或调侃
-> 发现它代表一类中年处境
-> 拉出身边人/旧经历对照
-> 给一个冷一点的现实判断
-> 短句收束
```

## 段落节奏

- 每段只推进一个意思，但不是一句一段。
- 数据段可以长一点，转折段可以短一点。
- 图片或截图前后要解释它证明什么。
- 同类清单合并成段，不要拆成机械列表，除非用户明确要清单。

## 结构避雷

- 不要“总分总”作文腔。
- 不要每节都用小标题，原文多数靠自然段推进。
- 不要结尾机械复述全文。
