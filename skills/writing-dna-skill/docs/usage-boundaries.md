# 使用边界

`writing-dna-skill` 用于写作风格研究、个人写作资产沉淀和 agent workflow 实验。

请注意：

- 不要在公开仓库中提交未经授权的原文语料。
- 不要使用本工具冒充作者本人。
- 不要用复刻文本误导读者相信其来自原作者。
- 如果分析对象包含付费文章、私密内容或非公开材料，请先确认你有处理和存储权限。
- 若输出用于公开发布，建议标注这是风格参考写作，而非原作者作品。

推荐用途：

- 分析自己的历史文章，沉淀个人 Writing DNA。
- 分析公开文章的写作结构和表达规律。
- 让 agent 在写作前读取风格文档，提高一致性。
- 对比不同作者在同一议题上的写法差异。
