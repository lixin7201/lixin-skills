# 素材使用规则

## 素材优先级

1. 强情绪故事：影视、新闻、普通人经历、朋友案例。
2. 熟悉生活场景：家庭饭桌、工位、聊天框、房间、消费、通勤。
3. 权威轻支撑：书籍、心理学观点、管理学概念、传统格言。
4. 数据和报告：只在能增强可信度时使用，不做主角。
5. 图片/截图：作为证据或节奏分隔，不在正文里硬解释图片内容。

## 嵌入方式

- 开头用一个故事打开情绪。
- 每节只放 1-2 个主材料，避免堆砌。
- 材料后必须抽象成判断句。
- 判断后必须落回普通人处境。
- 复杂材料只取与本文判断有关的一刀。

## 素材不足处理

如果缺少时间、地点、人物、价格、政策、活动信息、数据来源、采访来源、商家信息、权威说法，先列缺口，不写成事实。

可写降级版本：

- 关系/认知短评。
- 选题备忘。
- 补采问题清单。
- 只基于用户提供材料的观点稿。

## 不同稿件素材配置

- 家庭关系：影视/真实故事 + 心理机制 + 普通家庭动作。重点写关系如何从承担变成默认、从委屈变成控制、从控制变成反噬，最后落到边界和分工。
- 职场单位：同事/上司场景 + 一句职场法则 + 自保动作。自保动作要具体到确认、留痕、反馈、延期、责任边界、证据保存，不只写“成熟的人不内耗”。
- 金钱认知：消费/存款/工作场景 + 习惯因果 + 守钱动作。
- 清单方法：每节一个小故事或概念，不需要每条都长；但标题承诺了几条，正文就必须给够几条。
- 热点事件：只用已确认事实，不扩写未证实细节。暴力、法律、未成年人、家庭冲突等敏感题，只保留公共事实入口，不替当事人补前史或动机。
