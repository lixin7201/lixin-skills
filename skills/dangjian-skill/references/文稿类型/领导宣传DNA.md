# 领导宣传 DNA

## 适用场景

领导莅临调研、走访指导、座谈交流、慰问关怀、行业协会/部门/外地同行参观交流。

## 稳定结构

1. 标题：领导身份 + 莅临/到访 + 大宜宾 + 调研指导/交流。
2. 导语：时间 + 领导/单位 + 调研主题。
3. 接待：大宜宾/练团子相关负责人接待陪同。
4. 参观汇报：数据大屏、招聘、直播、妇联/工会/党支部建设、党建品牌。
5. 反馈：只写确认过的肯定或要求；不确定则写“认真听取、详细了解”。
6. 承接：以此次调研为契机，持续强化党建引领和业务规范。

## 写法要点

- 领导职务必须准确，顺序不能乱。
- 不要过度抒情，重点是调研主题、汇报内容、工作成效。
- 可以写“给予充分肯定”，但必须来自素材。
- 结尾常写“在指导下/以调研为契机/进一步”。
- 领导调研稿以紧凑为优先，常规调研控制在 5-6 段：到访导语、参观/介绍、座谈/汇报、交流反馈、下一步。
- 荣誉和业务数据只选与调研主题最相关的 2-3 个，不要把支部全部荣誉和所有业务板块都塞进去。
- 参访人员反馈要降级处理：素材只说“积极反馈/交流”时，写“围绕相关做法进行交流”，不要扩成“高度评价、参考价值突出”。

## 标题公式

- `某某领导一行莅临大宜宾调研指导`
- `某单位到大宜宾座谈交流 共促 + 主题`
- `组织关怀暖人心 + 慰问/调研 + 促发展`

## 反例

- 不核实职务就写完整头衔。
- 把领导宣传写成企业自夸。
- 写“作出重要指示”但素材没有这个表态。
- 借调研稿展示全部党建成绩，导致主线从“调研交流”跑成“综合汇报”。
