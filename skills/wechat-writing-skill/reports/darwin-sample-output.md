# Darwin 出稿样例

以下样例用于验证 Skill 的出稿流程，不来自 holdout 原文。

## 样例 1：加菲推断线 + 老街开放

【主标题】
封闭1个月后，宜宾老城区这条街重新开放了！

【副标题】
步道、灯光、休息座椅都有新变化，你还记得它以前的样子吗？

【正文】
这几天，路过老城区的宜宾人，应该已经注意到了这个变化。

此前围挡了一个月的这条老街，已经重新开放。现场能看到，路面步道、灯光和休息座椅都有了新变化，整条街看起来比之前更清爽。

对很多老宜宾来说，一条街的变化，往往不只是“修好了没有”这么简单。

它可能是小时候走过的路，是下班顺路买东西的地方，也是很多人对老城区最熟悉的一段日常。现在重新开放，最直观的感受就是：能走了，也更好停下来看看了。

目前能确认的信息，主要来自现场照片。至于后续是否还有新的规划、周边是否会继续调整，还要等进一步消息。

如果你最近也去过这里，欢迎在评论区补充现场变化。你还记得这条街以前的样子吗？

自检：

- 没有写加菲本人亲历。
- 没有补官方规划、历史年份或路线。
- 有地点对象、变化、普通人关系和评论入口。

## 样例 2：敏感维权线索安全改稿

【主标题】
宜宾一商家纠纷引关注，当事人希望了解后续处理方式

【副标题】
目前材料还不完整，合同、沟通记录和处理结果仍需进一步核实

【正文】
近日，有网友向我们反映，自己在宜宾遇到一件消费纠纷，希望通过平台寻求更多建议。

从目前提供的截图看，双方确实存在沟通争议。但由于完整合同、付款凭证、商家回应和调解结果还没有进一步确认，这件事暂时不适合直接定性。

对于类似情况，建议当事人先整理好以下材料：完整合同或订单记录、付款凭证、双方沟通截图、售后或协商记录，以及是否已经向相关部门反映。

如果后续有更完整材料，也可以继续补充。我们更希望把事实说清楚，把求助入口留出来，而不是在信息不完整时先下结论。

遇到类似消费纠纷的宜宾人，也欢迎在评论区说说自己的处理经验。

自检：

- 删除了“黑心商家”“曝光”等定性词。
- 没有补官方介入或商家责任。
- 改成求助、提醒和补证据口径。
