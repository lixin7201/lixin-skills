# 账号总风格

## 一句话定位

成都美食不是“高冷餐评”，而是本地吃货编辑部带路：把成都及周边的店、街、单品、老板和排队现场，写成读者马上能判断“值不值得去”的图文攻略。

## 读者默认画像

- 住在成都或常来成都，熟悉本地地名和方言语感。
- 关心性价比、排队成本、是否正宗、是否适合带朋友去。
- 喜欢“编辑部先去试了”的代入感，不需要学院派餐评。

## 账号共同手感

1. 标题先给具体对象或强反差：哪条街、哪家店、哪种单品、排队多久、谁在吃。
2. 开头先交代“为什么现在要看”：新发现、回访、朋友安利、编辑部反复去、某片区被低估。
3. 正文靠现场推进：位置/环境/老板/菜品/口感/价格或 Tips，少讲空泛道理。
4. 评价有口语判断，但必须绑定具体证据：汤底、火候、油辣、份量、排队、营业时间、街坊反馈。
5. 图片密度高，文字承担“带路”和“判断”，不抢图片的展示功能。
6. 结尾给实用入口：店名、地址/营业时间/避坑提醒，或让读者评论补充。

## 情绪强度

允许夸张、惊讶、撒娇式吃货语气，但不能空喊。每一次“太好吃”“绝”“值得”后面都要跟一个具体可感细节。
