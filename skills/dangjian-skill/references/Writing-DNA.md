# Writing-DNA

## 运行公式

`党建文章 = 组织身份 + 具体活动/荣誉/调研 + 民生业务落点 + 党建意义 + 下一步动作`

## 核心不是“官话”，而是“党建如何落到业务”

训练语料里的稳定写法，是把党组织建设、青年党员、民生服务、本土互联网平台、就业/婚恋/消费/政务传播等业务放在同一条链上：

1. 先确认组织站位：练团子党支部、四川练团子文化传播有限公司、大宜宾平台、新兴领域党组织。
2. 再交代真实动作：开展活动、参与学习、接受调研、获得荣誉、组织分享、服务群众。
3. 接着把动作转译成党建意义：党建引领、凝心聚力、服务民生、实干担当、同频共振。
4. 最后给下一步：持续深化、进一步强化、立足岗位、服务地方发展。

## 必读顺序

1. `账号总风格.md`
2. `账号选题判断框架.md`
3. `文章结构模板.md`
4. `原味指纹.md`
5. 对应 `文稿类型/*DNA.md`
6. `像不像判别器.md`
7. `去AI味保真补丁.md`

## 类型稳定度

- `日常活动开展`、`读书分享会、精英研学会`、`领导宣传`：样本充足，可作为稳定类型。
- `发言稿`、`获奖宣传`：样本少于 10 篇，只作为原型类型；写作时更依赖用户事实素材，不做 95% 高保真承诺。

## 输出边界

这套 Skill 适合“练团子党支部/大宜宾平台风格的党建稿”。如果用户要写其他单位党建稿，保留写法，但必须替换组织事实、业务事实、荣誉事实和口号体系。
