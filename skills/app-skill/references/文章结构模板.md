# 文章结构模板

## APP 帖通用结构

```text
【标题】具体对象 + 情绪/利益/风险/动作
开头：一句话给出事件和用户关系
事实：谁、哪里、什么时候、发生了什么
展开：官方/网友/现场/名单/步骤/背景
判断：这事为什么值得关注，边界是什么
行动：绕行/报名/核实/收藏/联系/提醒
评论口：问一个用户真能回答的问题
```

## 结构路由

- 网友曝光：网友线索 -> 已确认事实 -> 待核实点 -> 同类经历征集。
- 城建交通：变化对象 -> 位置/范围/时间 -> 对生活影响 -> 评论补充。
- 政务公告：来源 -> 核心变化 -> 影响人群 -> 办理/注意事项。
- 招聘求职：单位/岗位 -> 人数/条件 -> 薪资福利 -> 报名方式 -> 风险提示。
- 突发应急：最新时间点 -> 已确认情况 -> 安全提醒 -> 持续更新入口。
- 本地生活：第一现场 -> 怎么玩/怎么去/适合谁 -> 注意事项 -> 评论补充。
- 公益求助：当事人情况 -> 真实困难 -> 已有证明/渠道 -> 帮助方式。

## 段落节奏

APP 允许短段，但短段必须承担功能：强调、转折、步骤、提醒、评论口。连续 5 段以上同功能短句要合并。
