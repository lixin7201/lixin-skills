# 槽值 Writing DNA

## 调用公式

账号人格 + 互动验证过的题材路线 + 事实/网友材料 + 槽值式反差判断。

## 最小可执行写法

1. 先判断选题属于哪条路线：综合热点、消费品牌、地域迷因、娱乐人物、社会情绪、身体审美、日常冷知识、公共教育或怀旧 IP。
2. 标题必须有“具体对象 + 反常识/冲突/羞耻感/荒诞感”，不能只写主题。
3. 开头 3-6 段完成：爆点事实、读者第一反应、问题翻面、本文要拆的真矛盾。
4. 正文按“现象 -> 证据 -> 反差 -> 更深原因 -> 人群处境/时代判断”推进。
5. 每个判断后接材料：平台热评、数据、品牌/行业事实、历史脉络、生活场景或影视/社交截图。
6. 槽值可以毒舌，但毒舌对象是现象、话术、商业逻辑和集体困境，不凭空辱骂具体个人。
7. 结尾不要喊口号，落到一个可复述的判断：这件事为什么让人好笑、难堪、心酸或清醒。

## 像槽值的关键

- 选题不是“发生了什么”，而是“这件小事暴露了什么集体情绪/消费神话/审美规训”。
- 文字要有年轻人的第一反应：惊讶、吐槽、阴阳怪气，但很快进入事实拆解。
- 常用强对比：曾经高端/现在不灵，明明荒诞/却很真实，看似笑话/其实刺痛。
- 能把社交平台的热闹，翻译成一代人的处境。
- 每篇都要有“槽”：槽的不是猎奇，是那些被包装过的虚假体面。

## 不像的常见原因

- 开头像通稿或百科介绍，没有第一反应和冲突。
- 只有情绪输出，没有事实链、网友材料或行业背景。
- 句子全是 AI 式工整并列，缺少短促转折、冷判断和反讽。
- 结尾强行升华成鸡汤，而不是回到具体处境。
