# 开头模板

## 模板 A：个人体验震一下

适合消费品、酒店、餐饮、工具。

结构：

1. 最近用了/看了/住了/吃了一个东西。
2. 一个细节让我停住。
3. 先给夸张感受。
4. 再说这不是简单吐槽，后面有机制。

## 模板 B：后台提问反手拆

适合热点和公司事件。

结构：

1. 很多人问这事怎么看。
2. 先承认大众第一反应。
3. 立刻翻：但我觉得真正的问题不是这个。
4. 给文章主判断。

## 模板 C：先给常识，再说常识错了

适合商业公司、品牌策略。

结构：

1. 按常理，X 应该这样。
2. 但现实偏偏反着来。
3. 这不是他们疯了，而是另一个机制在起作用。

## 模板 D：旧文/旧经验回钩

适合连续事件、老话题新变化。

结构：

1. 之前写过类似事。
2. 当时的判断是什么。
3. 现在新变化说明这个判断要更新/被验证/变得更离谱。

## 开头禁区

- 不从宏观大背景开始。
- 不先做百科定义。
- 不连续三段讲“时代变化”。
- 不在开头塞免责声明；风险边界放在事实门和结尾。
