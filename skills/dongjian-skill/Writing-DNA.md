# 洞见 Writing-DNA

## 一句话定位

洞见式文章把普通成年人的焦虑、委屈和困惑，转译成一个强判断：你痛苦不是因为单个事件，而是因为关系、认知、边界、习惯或心气出了问题。

## 选题 DNA

优先写中年普通人高频痛点：家庭、职场、金钱、人际、内耗、身体、孩子、关系边界。素材可以是新闻、影视、名人、书籍、普通人故事，但目的不是报道，而是解释读者自己的生活。

## 标题 DNA

标题必须有判断，但判断先来自素材，不先来自公式。常见骨架：

- `一个人/一个家庭最大的 X：Y`
- `人在单位：A、B、C`
- `为什么 X？答案在 Y`
- `普通人变强/变富的开始：Y`
- `中年以后才懂：X`
- `看了 X，才知道 Y`

这些骨架是候选，不是填空题。连续生成多个标题时，要让切入点变化：有的抓关系失衡，有的抓行动代价，有的抓普通人的自救动作，有的抓一处具体细节。

## 开头 DNA

开头 3-8 段进入主题。常见顺序：

1. 一句价值判断或金句。
2. 一个熟悉故事、热搜、影视、朋友经历。
3. 抛出核心困惑。
4. 把困惑命名成本文主判断。

## 结构 DNA

主流结构是数字分节，3 节最多，4-5 节用于清单/方法稿。每节内部用：案例进入 -> 抽象判断 -> 读者处境 -> 行动/警句。

结构稳定不等于段落模板固定。每篇要按素材选择推进重心：有的先写关系变化，有的先写生活失序，有的先写一个反常细节，有的先写行动清单。不能把所有题都写成同一种三段论。

## 语言 DNA

- 句长平均约 25 个中文字符，短句占明显比例；短句用于判断、停顿和姿态，故事链和解释链不要一项一段。
- 喜欢对偶和递进：`你怎样...，生活就怎样...`、`不是...，而是...`、`越...，越...`。
- 高频气质词：清醒、体面、边界、心气、内耗、认知、关系、普通人、中年、成年人。
- 金句要可转发，但不能密到像语录集合。

## 素材 DNA

素材是情绪入口和证据，不是堆砌。优先顺序：熟悉场景/故事、影视/书籍/名人、心理学或管理概念、社会新闻、普通人观察。硬事实不全时，宁可写关系判断，不写具体指南。

## 转折 DNA

常用推进：

- 表面问题 -> 深层原因。
- 个体故事 -> 普遍人性。
- 情绪共鸣 -> 清醒判断。
- 被伤害 -> 设边界。
- 抱怨 -> 行动。

## 结尾 DNA

结尾通常在 `▽` 前：回扣标题，给一组短句，最后落到推荐/点赞或行动。结尾不是总结全文，而是把读者推向一种姿态：腰直、心正、嘴稳、边界清、少内耗。

## 不像的信号

- 只有温暖口吻，没有强判断。
- 只有新闻细节，没有人生转译。
- 概念密集，故事太少。
- 标题、开头和分节像同一个填空模板换词。
- 不同题材都用同一条判断链，缺少素材自身的转折。
- 结尾只是“希望大家越来越好”。
- 编造事实、价格、政策、采访原话。
