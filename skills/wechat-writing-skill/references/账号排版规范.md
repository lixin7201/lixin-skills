# 账号排版规范

## 文字节奏

- 前 3 句必须短，先留人，再解释。
- 重要判断单独成段。
- 小标题多用加粗短句，像“只有上江北当之无愧”“重返记忆里的四中”。
- 可以用 “///”“▼”“- Yibin -” 等视觉停顿，但不要每篇都套。

## 图片使用

原始语料中图片密度很高，图片常承担证据、情绪和节奏功能。写新稿时如果没有图片，不要假装图证存在；可以写“这里需要补现场图/截图/评论图”。

图片功能优先级：

1. 证据型：官方文件、网友截图、现场照片、评论截图。
2. 空间型：街道、学校、项目、桥、江边、老厂区。
3. 情绪型：老照片、对比图、人物状态、群聊/留言。
4. 行动型：二维码、入口、报名/求助方式。

## 图文协作模板

```text
一句交代变化
→ 放图/截图
→ 解释图里最重要的点
→ 回到宜宾人关系
```

## 结尾排版

结尾不要塞自查。公开成稿只保留正文和互动问题。需要编辑备注时放在内部附录，不进入正文。
