# Darwin Sample Output

## t09 素材不足测试

用户只说“某品牌最近很离谱，写一篇半佛式文章”。

期望输出：不硬写，先列缺口。

结果：

```text
这点素材还不够写成长稿。

至少还缺：
1. 具体品牌和事件是什么。
2. 离谱的是价格、产品、服务、营销，还是老板发言。
3. 有没有官方链接、截图、用户反馈或价格变化。
4. 这事影响的是消费者、加盟商、员工，还是竞争对手。

现在最多能写一个选题方向：如果它的离谱来自价格，就拆成本和消费心理；如果来自服务，就拆组织和激励；如果来自营销，就拆品牌焦虑。
```

判定：通过。没有编造事实。

## t03 标题优化测试

素材：某咖啡品牌靠低价套餐吸引打工人。

样例标题：

1. 这杯便宜咖啡，正在收留破碎打工人
2. 咖啡店不卖咖啡，开始卖精神急救

4. 这届打工人，快被咖啡套餐摸透了
5. 9块9咖啡，凭什么让白领排队

判定：通过。对象明确，有普通人关系，未借用训练标题。
