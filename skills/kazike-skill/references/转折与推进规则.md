# 转折与推进规则

## 常用推进链

```text
看到现象
-> 第一反应
-> 为什么这不是普通现象
-> 放证据
-> 解释证据
-> 转到普通人关系
```

## 常用转折

- `但真正离谱的，不是这个。`
- `更有意思的是。`
- `问题也在这里。`
- `酷归酷，但我还是要说一句。`
- `这件事最值得警惕的地方是。`
- `如果只看到这里，那还是低估它了。`
- `所以，这件事表面是 X，本质是 Y。`

## 小节推进

每个小节至少完成一个变化：

- 从现象到原因。
- 从功能到场景。
- 从爽点到坑点。
- 从个人体验到普通人关系。
- 从单个产品到行业趋势。

## 禁忌

- 不要只靠“此外/另外/同时”平铺资料。
- 不要把每一节都写成同样长度。
- 不要没有转折就直接上价值。
