# 素材使用规则

## 素材优先级

1. 用户提供的一手事实、截图、链接、数据。
2. 官方公告、财报、产品页、公开采访。
3. 多家媒体交叉验证的事实。
4. 用户评论、社交平台讨论，只能用于呈现情绪和场景，不能当事实结论。
5. 作者个人体验，只能在用户明确提供时使用。

## 半佛式素材动作

- 先把素材讲成人话。
- 再把人话翻成机制。
- 价格、数量、时长、门店、功能、限制条件要尽量具体。
- 图片/截图出现后必须解释它证明什么，不能只说“如图”。
- 如果素材是争议事件，必须区分事实、当事人说法、网友情绪和作者推断。

## 缺素材时的处理

- 缺时间地点：不写成新闻事实。
- 缺数据：不用“暴涨、暴跌、很多、无数”做硬结论。
- 缺来源：写成“如果这个说法属实”，并列出需要补的证据。
- 缺用户体验：不能伪造“我试了一下”。
