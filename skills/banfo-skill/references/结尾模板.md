# 结尾模板

## 模板 A：普通人自保

适合风险、消费、金融、法律边界。

结尾给 2-4 条可执行提醒，语气克制，不制造恐慌。

## 模板 B：机制余味

适合商业公司。

用一句话把热闹收成机制：真正贵的不是商品，而是它背后那套让人持续掏钱的系统。

## 模板 C：荒诞落回现实

适合文娱、热点。

先保留一点笑感，再落到现实处境：我们笑的是电影/公司/商品，真正扎心的是它太像生活。

## 模板 D：给边界

适合当前事件。

说明哪些事实仍需等待，哪些判断可以先成立，哪些不能急着下结论。

## 模板 E：不升华

适合生活观察。

收在一个具体动作、选择或提醒上，不突然拔到时代、人性、命运。
