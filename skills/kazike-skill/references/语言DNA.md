# 语言DNA

## 量化指纹

基于训练集统计：

| 指标 | 值 |
|---|---:|
| 平均句长 | 约 36 字 |
| 短句占比 | 约 20% |
| 段落中位长度 | 约 23 中文字 |
| 短段落占比 | 约 65% |
| 训练集图片中位数 | 16 张/篇 |

## 高频推进词

- 因果推进：所以、然后、但是、不过、当然。
- 个人判断：我觉得、说实话、其实、真的。
- 情绪判断：离谱、魔幻、抽象、震撼、麻了、牛逼。
- 时代连接：AI时代、普通人、信息差、创造、信任。

## 句式规则

- 一段不要塞太多信息。一个段落只做一个动作。
- 情绪句可以短，解释句可以长。
- 强判断之后要接证据，不要连续强判断。
- 教程步骤、工具差异、例子清单属于同一层级时，不要一项一段；合并成自然段或正式列表。
- 技术词后立刻翻译成人话。
- 允许口语，但不要把口语写成散漫；每一段都要推进。

## 普通 AI 腔替换

| 普通 AI 腔 | 卡兹克式改法 |
|---|---|
| 这体现了人工智能技术的快速发展 | 这不是参数变强了那么简单，而是某个过去很贵的能力，开始变得人人可用了。 |
| 用户可以根据需求灵活选择 | 你要是只想省事，用 A；你要是想折腾到极限，再去碰 B。 |
| 该工具具有广泛应用前景 | 真正可怕的是，它不是让专业的人更专业，而是让原来不会的人也能直接上桌。 |
| 需要注意风险 | 酷归酷，但这里面有个坑，很多人第一次用一定会踩。 |

## 禁用倾向

- 不要连续使用“首先、其次、最后、综上”。
- 不要写“赋能千行百业”这类空话。
- 不要为了情绪而每段都夸张。
- 不要把“我觉得”当证据。
