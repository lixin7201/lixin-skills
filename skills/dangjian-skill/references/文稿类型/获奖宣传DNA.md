# 获奖宣传 DNA

## 适用场景

党组织获奖、先进基层党组织、星级党组织、两优一先、党建品牌案例、荣誉喜报。

## 稳定结构

1. 喜报式标题：先把荣誉讲清楚。
2. 荣誉意义：上级认可、权威印证、责任鞭策。
3. 长期积累：支部成立、党员队伍、组织架构、党建品牌。
4. 业务融合：就业、婚恋、消费、政务传播、公益服务等板块。
5. 公益担当：爱心送考、助农、关爱、反诈等行动。
6. 下一步：珍惜荣誉、再接再厉、做优品牌。

## 写法要点

- 情绪可比日常简报更昂扬，但不能夸大荣誉等级。
- “荣誉不是终点”是常见收束，但必须接具体行动。
- 样本不足 10 篇，写作时优先依赖用户提供的荣誉文件和事实。

## 标题公式

- `喜报！主体荣获 + 荣誉称号`
- `荣誉/星级 + 筑梦/谱新篇 + 党建品牌`
- `品牌名 + ：党建引领 + 民生服务 + 新实践`

## 反例

- 不写表彰单位和荣誉名称。
- 把荣誉宣传写成普通活动简报。
- 连续堆“荣光、闪耀、辉煌”，没有组织建设和业务成效。
