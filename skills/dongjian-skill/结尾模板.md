# 结尾模板

以下是收束路径，不是要输出的段落标签。正式正文里不要写“结尾”两个字当小标题；直接用短判断、`▽`、推荐语或自然段落完成收束。

## 1. 回扣标题式

适合：所有观点稿。

```text
人这一生，谁都有 [困境] 的时候。

但越是在这种时候，越要守住 [本文关键词]。

[短句一]；[短句二]；[短句三]。

点个推荐，愿你 [姿态/行动/结果]。
```

## 2. 三短句收束

适合：金钱、体面、低谷。

```text
再难，别丢了心气。
再穷，别丢了骨气。
再累，也别把自己过成讨厌的样子。
```

## 3. 边界提醒式

适合：关系、家庭、付出。

```text
好的关系，从来不是一个人拼命扛。

懂得停下来，懂得说不，懂得把自己放回生活里，才是真正长久的爱。
```

## 4. 行动号召式

适合：清单、方法。

```text
别等到一切都准备好，再开始改变。

先从今天最小的一件事做起：收拾房间、早睡一次、少回一句、存下一笔小钱。
```

## 5. 留白式

适合：人物/影视/故事稿。

```text
很多答案，日子会慢慢告诉你。

只是到那时你会明白：真正救你的，从来不是别人回头，而是你自己醒来。
```

## 6. 评论引导式

适合：争议较轻的关系话题。

```text
你身边有没有这样的人？

点个推荐，也提醒自己：别在错的人身上，耗尽好心气。
```

## 不像的结尾

- “以上就是本文全部内容。”
- “希望本文对你有所帮助。”
- “让我们共同努力建设美好未来。”
- 正文里出现“结尾”两个字当段落标签。
- 只升华，不回扣具体主题。
- 只让读者转发点赞，没有价值收束。
