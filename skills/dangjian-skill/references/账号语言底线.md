# 账号语言底线

## 可以用的原味词

- 党建引领、红色引擎、凝心聚力、实干担当、初心使命、服务民生、赋能发展。
- 新兴领域、互联网企业、数字经济、本土平台、党群联动、党建带群建。
- 同频共振、双向赋能、走深走实、落地见效、提质增效、示范引领。
- 主题党日、组织生活、党员骨干、业务一线、群众需求、社会责任。

## 标题常用骨架

- `党建引领 + 业务/活动 + 成效`
- `红色词 + 行动词 + 发展/民生词`
- `喜报 + 主体 + 荣获 + 荣誉`
- `领导身份 + 莅临/到 + 大宜宾 + 调研指导/座谈交流`
- `组织第X期 + 读书分享会/精英研学会`

## 必须避免

- “在这个快节奏时代”“让我们一起见证”“未来可期”“满满干货”等通用公众号腔。
- “根据你提供的素材”“资料显示”“原文中写道”等生成过程泄露。
- 不核实就写“高度赞扬、充分肯定、重要指示、圆满成功、成效显著”。
- 不加事实就连续堆“赋能、温度、力量、担当”。
- 把所有稿子都写成“为深入贯彻...活动中...下一步...”一个模子。

## 句式节奏

- 简报类：多数段落 80-180 字，允许标题/图片说明/简报头形成短段。
- 发言稿：长段可接受，常用“一、二、三”和“（一）（二）（三）”。
- 获奖宣传：可用较长段展开组织建设、品牌、成效。
- 领导宣传：段落要短稳，事实密度高，少抒情。
