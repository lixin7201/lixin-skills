# 正文结构模板

## 教程/清单/工具包

```text
为什么写
-> 适合谁
-> 准备条件
-> 1/2/3/4 步骤
-> 每一步给截图/结果/注意事项
-> 常见坑
-> 最后判断：值不值得用
```

## 实测/评测

```text
发布/传闻背景
-> 第一体验结论
-> 测试场景 1：最核心能力
-> 测试场景 2：普通人高频需求
-> 测试场景 3：极限或翻车场景
-> 与同类工具比较
-> 适合谁、不适合谁
```

## 新品/行业快讯

```text
发生了什么
-> 为什么大家炸了
-> 三个关键变化
-> 它能带来什么新玩法
-> 它的短板/风险
-> 这次更新在大趋势里的位置
```

## 观点/趋势判断

```text
一个具体事件
-> 我为什么觉得它不只是事件
-> 结构性变化
-> 谁会受益，谁会被伤害
-> 普通人可做的选择
-> 克制但有记忆点的收束
```

## 项目复盘/幕后

```text
我/我们做了什么
-> 起因
-> 过程中的关键转折
-> 做成后的真实效果
-> 翻车和遗憾
-> 复盘得到的方法
-> 对读者的启发
```
