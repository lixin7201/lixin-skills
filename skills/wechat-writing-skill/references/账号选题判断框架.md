# 账号选题判断框架

## 五步判断

1. 对象是否具体：人、地点、项目、学校、企业、岗位、事件、时间至少有一个。
2. 点开收益是否明确：消息、机会、避坑、共鸣、长脸、记忆、站队、帮忙。
3. 素材是否支撑体量：强素材写完整公众号；中素材写观察/服务；弱素材写短提醒或征集。
4. 风险是否可控：涉及医疗、法律、未成年人、维权、隐私、政策待遇，缺权威来源就降级。
5. 评论区是否有话可接：能让读者补充经历、记忆、线索、态度，才适合大宜宾公众号。

## 素材强弱

| 等级 | 条件 | 稿件形态 |
|---|---|---|
| 强 | 官方信息/现场图/截图/评论/采访/明确数字中至少 3 类 | 完整公众号稿 |
| 中 | 有明确对象和 2 个事实点，但缺现场或关键补充 | 保守观察稿、服务提醒 |
| 弱 | 只有一句线索、传闻、单张图、非权威转述 | APP 话题、短提醒、线索征集 |

## 标题优先级

1. 对象前置：宜宾某地、某人、某项目、某学校先出现。
2. 变化清楚：再见、开通、开放、拆迁、火了、来了、公开招聘。
3. 情绪有钩：惊喜、舍不得、气愤、心疼、长脸，但不能替代事实。
4. 数字慎用：只有来源明确时用金额、人数、面积、时间。

## 选题打回

- 只有外部通稿，没有宜宾普通人关系。
- 只有小红书/视频热度，没有本地事实核验。
- 只有网友单方控诉，要求写曝光定责。
- 只有活动宣传语，缺时间地点入口。
- 只有项目愿景，缺位置、阶段、官方来源。
