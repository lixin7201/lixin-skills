# 日常活动开展 DNA

## 适用场景

主题党日、学习培训、清明祭扫、志愿服务、公益助农、观看节目、联建活动、福利发放、党课参与。

## 稳定结构

1. 简报头和活动标题。
2. 第一段：时间 + 活动背景/目的 + 参与主体。
3. 第二段：现场流程和关键动作。
4. 第三段：代表发言、学习内容或参与感受。
5. 末段：活动意义 + 下一步。

## 写法要点

- 字数多在 450-800 字，短而完整。
- 必须有具体活动流程，不能只写意义。
- 党建意义放在事实之后，不抢第一段。
- 结尾可用“下一步”，但要接活动带来的具体承接。

## 标题公式

- `党建引领 + 活动主题 + 促发展/显担当/暖人心`
- `练团子党支部开展 + 主题 + 活动简报`
- `红色/公益/学习关键词 + 动作 + 党建意义`

## 原味动作词

开展、参与、组织观看、集中学习、联合、慰问、助力、协办、受邀、收看、走访。

## 反例

- 不写时间地点人物，直接“此次活动意义重大”。
- 把小活动写成宏大理论文章。
- 只罗列流程，没有支部承接和下一步。
