# 结尾模板

## 模板 1：现实提醒

```text
所以，对普通家庭来说，[动作]之前先想清楚自己的现金流、孩子、身体和退路。
别上头，别跟风，能少折腾一点就少折腾一点。
```

## 模板 2：一家之言

```text
以上只是我个人经验的归纳，不一定准确，算是抛砖引玉。
大家有不同看法，评论区聊聊。
```

## 模板 3：命运感

```text
很多事情回头看才知道，可能就是时代和命运。
普通人能做的，就是别把自己逼到没有退路。
```

## 模板 4：自嘲落地

```text
行了，就写到这吧。
我现在的想法很简单，少折腾，保住身体，保住家里这点日子。
```

## 结尾禁忌

- 不要“愿每个人都能找到属于自己的答案”。
- 不要突然升华成民族、时代、文明。
- 不要给确定承诺。
- 不要机械总结三点。
