# 视觉风格指南

## 图文节奏

训练集平均图片约 16 张/篇。图片通常承担证据功能，不只是配图。

常见节奏：

```text
短说明
-> 截图/生成结果
-> 解释图片证明了什么
-> 下一张图或下一步操作
```

## 图片类型

- 界面截图：证明入口、设置、步骤。
- 生成结果：让读者先感受效果。
- 对比图：证明模型差异。
- 社媒截图：证明热度或争议。
- 视频封面：用于视频能力或案例展示。
- 表格/榜单：辅助判断，不替代体验。

## 排版规则

- 段落短，空行多；判断和图片承接可短，教程步骤、工具差异和证据链不要拆成连续微段。
- 小标题常用加粗或编号，不追求复杂层级。
- 教程/清单用 `一. 二. 三.` 或明确小节标题。
- 重点句可以加粗，但不要整篇到处加粗。
- 链接单独成行，方便读者复制。

## 没有图片时怎么写

如果用户没有给图，不要伪造“截图如下”。改成：

- 标注“这里适合放界面截图/生成结果/对比图”。
- 用文字描述图应证明什么。
- 对教程类，提醒用户补 3-5 张关键图。
