# 公共教育/事件纪念 DNA

- 训练样本：15 篇
- 定位：教育、医疗、公共事件、人物离世或社会纪念。
- 推进链：事实先行 -> 克制反应 -> 个体故事 -> 公共意义 -> 不煽情收束

## 标题

必须给具体对象和反差，不写“关于 X 的思考”。优先使用“第一批 X”“X 怎么就没人骂”“千万别 X”“这下，X 真尴尬了”等槽值式强入口。

## 开头

1. 先抛最新事实或最离谱的一幕。
2. 写读者第一反应。
3. 用一句话翻面：真正值得看的不是热闹本身，而是它暴露的逻辑。

## 正文材料

- 至少 3 组材料：平台讨论/网友现场、数据或历史变化、行业/人物/城市背景。
- 每组材料后必须给判断，不堆资料。
- 如果用户没有给事实，先列待核实，不编。

## 结尾

回收为一句冷判断或生活处境，不要鸡汤。

## 高分样例标题

- 霍金去世，轮椅太小，但恰好宇宙够大
- 李咏患癌去世：生命来来往往，来日并不方长
- 偷窥大学生秘密瞬间，我一秒破防
- 高考差一分过线的人，后来都怎么样了
- 英国女王去世，伦敦桥倒了
