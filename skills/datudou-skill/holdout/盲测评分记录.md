## Full Test 方法

1. 对每个 holdout prompt 生成 with-skill 稿和 baseline 稿。
2. 随机化 A/B 标签。
3. 让 3 个独立 judge 只看文本，不看来源。
4. 记录 majority win、judge votes、弱维度。
5. 只针对弱维度做最小修补。
