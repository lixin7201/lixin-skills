# 刘润公众号公开表达：语言 DNA

> 方法：按 `writing-dna-skill` L1-L2 提取。统计对象为 409 篇训练语料；未使用 holdout 验证集。本文描述“刘润公众号公开文章风格”，不是刘润本人授权口吻。

## 1. 语言 DNA

### 高频概念

训练统计中高频概念集中在这些词簇：

| 词簇 | 高频词 | 写作功能 |
|---|---|---|
| 商业底层 | 商业、价值、成本、价格、需求、效率、客户、交易 | 把现象还原为交换、成本和价值链 |
| 组织管理 | 管理、老板、员工、团队、目标、组织、责任 | 把人的问题翻译成系统设计问题 |
| 思考方式 | 问题、认知、判断、模型、本质、选择、简单、复杂 | 引导读者从观点进入方法 |
| 技术变化 | AI、软件、数据、系统、生产率、流程 | 解释技术如何改写价值链 |
| 人性长期 | 时间、信任、人性、成功、失败、机会 | 把商业判断落回人的行为和长期博弈 |

使用规则：不要只堆“底层逻辑、本质、认知”。每个抽象词后必须接一个可观察机制：谁、付出什么、得到什么、链路哪里变化。

### 句长和段落节奏

- 训练语料段落数：31792。
- 段落长度中位数：29 字符，均值：41 字符。
- 句子数：47981。
- 句长中位数：20 字符。
- 短句（<=15 字）占比：37.8%。
- 长句（>=50 字）占比：11.7%。

节奏特征：

1. 多短段落，一句一段会出现，但主要用于停顿、提问、反转和金句；事实链、案例链、机制解释链通常要合并成自然段。
2. 用短句制造停顿，例如“不是。”“怎么办？”“这很重要。”
3. 关键判断前常用连续提问铺垫。
4. 复杂解释里穿插口语短句，避免学术腔。
5. 段落之间留大量空行，阅读像“边讲边想”。

段落回归护栏：训练语料段落中位约 26-31 字。生成长文时，如果普通正文里 <=20 字短段超过一半，或连续 5 段以上都低于 20 字，优先检查是否把同一层级的排比、清单、例子和方法步骤拆碎了。

### 口头禅和常用句式

- “什么意思？”
- “怎么办？”
- “为什么？”
- “不是。是……”
- “我给你举个例子。”
- “说白了。”慎用，容易 AI 腔，只有解释口语化时使用。
- “这很重要。”
- “真正的 X，不是 Y，而是 Z。”
- “A 是目标，B 才是路径。”
- “如果你……，那你要先问……”

### 标点习惯

训练统计中逗号、句号、引号、顿号、问号和冒号高频。

- 问号多：用于连续拆问题，不只是标题悬念。
- 引号多：用于包装概念、引用流行说法、框定关键词。
- 冒号多：用于定义和转折后的解释。
- 破折号少于括号和引号，少用长插入句。
- 感叹号少量使用，主要用于情绪收束或行动提醒。

## 2. 标题 DNA

训练标题 409 个，其中含数字标题 140 个，占 34.2%；问题式标题 69 个，占 16.9%；含引号标题 50 个，占 12.2%。

常见标题结构：

1. 数字清单型：`X 条/种/件/个 + 清醒认知/建议/方法/能力`。
2. 问题拆解型：`为什么 X？`、`X 到底怎么赚钱？`、`该不该 X？`。
3. 反常识判断型：`不是 X，而是 Y`、`X 被低估了`。
4. 场景刺痛型：`润总，我不想去公司了`、`当没人敢说真话时……`。
5. 概念包装型：给现象起一个可传播概念，如“低信任行业”“能力半径”“原始资本”。
6. 公司/人物案例型：`复盘 X`、`X 为什么被惹毛了`、`和 X 聊完，我看见了……`。

标题不要写成普通 AI 标题，如“关于某某问题的深度分析”。要有一个明确钩子：数字、疑问、冲突、人物、反常识或可传播概念。

## 3. 开头 Hook

训练统计中的 hook 类型分布：观点引入式 140、数字清单式 99、问题式 69、反常识/定义式 57、场景/故事式 44。

可复用 hook：

1. 问题式：从读者正在问的问题进入，马上承认“这是个好问题”。
2. 场景式：先放一个饭桌、会议、客户、老板、读者提问、参访见闻。
3. 反常识式：先否定一个流行说法，再解释“问题不在这里”。
4. 概念定义式：先问“到底什么是 X”，再区分目标、路径、手段、机制。
5. 事件切入式：从热点公司或公共事件进入，但很快转到底层机制。

## 4. 结尾方式

常见结尾不是宏大口号，而是回扣核心概念后给一个行动或态度：

- 回扣定义：再次说明“所以，X 的本质是 Y”。
- 行动建议：给 3 条小建议或一个优先动作。
- 价值升华：从商业回到人、选择、信任、长期主义。
- 问题留白：留一个读者可以带走的问题。
- 祝愿式收束：适合个人成长和年度复盘题材。

结尾禁忌：不要用“综上所述”。不要把复杂判断硬收成鸡汤。不要冒充刘润本人说“我希望你”。
