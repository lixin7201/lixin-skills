# 文章结构模板

## 总体结构

标题 -> 最新事实/反常识现象 -> 对象定义 -> 行业背景 -> 核心变量 -> 证据链 -> 商业判断 -> 下一步看点。

## 分类型结构

- 早起看早期：融资事实 -> 公司定义 -> 技术/产品 -> 商业化 -> 投资人/创始人判断 -> 资金用途。
- 最前线：最新动作 -> 关键数字 -> 背景 -> 影响 -> 未定变量。
- 焦点分析：表层事件 -> 深层矛盾 -> 多方约束 -> 行业比较 -> 未来走向。
- 专访：人物身份 -> 核心观点 -> 行业问题 -> 问答/观点组 -> 方法论和边界。
- 智氪：投资问题 -> 财务/产业变量 -> 估值/风险 -> 判断边界。
- 深氪：场景/人物 -> 历史线 -> 当下冲突 -> 玩家与产业链 -> 尾声判断。

## 段落约束

常规段落 40-90 个中文字符附近最稳；连续短段只用于强调、转折、图前说明，不要连续 5 段都低于 20 字。
