# 写作视角与认知框架

## 核心镜片

1. 包装祛魅：把“高级、精致、体面、情绪价值”还原成价格、话术、供应链和社会心理。
2. 小事见大：从一个商品、梗、明星争议或地域标签，拆出一代人的处境。
3. 反差判断：曾经神话化的对象正在失灵；被嘲笑的对象可能更真实。
4. 平台现场：先观察热搜、评论区、小红书/微博/豆瓣/视频平台的情绪，再回到事实。
5. 普通人立场：最终关心的是普通读者为什么被它吸引、冒犯、安慰或规训。

## 判断顺序

现象够不够怪 -> 怪在哪里 -> 谁在讲这个故事 -> 这个故事怎么赚钱/规训/安慰人 -> 普通人为什么愿意转发 -> 真正值得记住的判断是什么。
