# 账号选题判断框架

## 先问四个问题

1. 这件事和承德的哪类人直接相关？没有具体人群，不写。
2. 它只是消息，还是能解释一个城市变化、公共问题或生活成本？只能转述消息时，降级为短提醒。
3. 素材里有没有不可编造的事实锚点：时间、地点、人物、数字、现场、官方口径、截图或链接？没有就不写长文。
4. 读者看完会得到什么：避坑、服务、理解、情绪释放、行动路径，至少要有一个。

## 可写选题

- 承德公共服务、交通、停车、住房、消费、教育、医疗等生活问题。
- 承德文旅、城市品牌、商业机会、活动赛事和产业变化。
- 普通承德人的真实困境、选择和反击。
- 外界如何看承德，以及承德如何被误解、被看见、被低估。

## 慎写选题

- 只有传闻没有来源的维权、事故、医疗和未成年人议题。
- 只有商家自夸，没有价格、体验、客流、授权或真实场景的推广。
- 只靠情绪骂人，无法给事实和公共价值的争议。

## 路由判断

- 生活痛点优先走民生稿。
- 官方政策/规划/公告优先走政务稿或服务提醒。
- 时间地点明确、读者可参与，走活动稿。
- 商家、品牌、餐饮、产品、工厂、消费，走商家稿。
- 有人物经历和转折，走人物稿或情感稿。
- 以城市命运和公共判断为主，走评论观点。
