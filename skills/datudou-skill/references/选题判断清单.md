# 选题判断清单

## 强势题材

1. 北京楼市：二手房、新房、学区房、老破小、总价段、土地供应、渠道和佣金。
2. 北京教育：高考、中考、小升初、区位差异、学区房、学科选择、职高退路。
3. 中年生活：身体、现金流、家庭责任、父亲工具人、提前退休、别折腾。
4. 年轻人和代际：不留北京、就业、婚育、生活方式、按部就班失效。
5. 家庭账本：中产线、养娃成本、房产净资产、消费降级、租售比。
6. 反常识社会观察：高门槛岗位、热门专业、周期变化、命运感。

## 弱势题材

- 纯产品推广、品牌软文、娱乐八卦、严肃政策解读、非北京本地民生。
- 没有数字、没有生活场景、没有普通家庭关系的宏观观点。
- 需要一手调查但用户只给标题的硬新闻。

## 选题变成文章的三步

1. 找入口：最近新闻、粉丝问题、自己经历、朋友案例、家庭小事。
2. 找账本：钱、分数、房价、时间、身体、家庭责任，至少找一个。
3. 找落点：保守、自保、少折腾、别跟风、接受周期、保住选择权。

## 标题前检查

- 能不能一句话说清“跟谁有关”？
- 能不能看出一个明确对象：北京、孩子、中年人、年轻人、房子、学校？
- 有没有可验证的现实材料，而不是纯感慨？
- 写完后读者能少踩一个坑，还是只获得一段情绪？
