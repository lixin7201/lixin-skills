# 标题DNA

## 标题原则

标题必须让读者同时知道三件事：对象是谁、情绪是什么、为什么现在值得点开。

## 常用标题公式

1. `一手体验 X - Y`
2. `实测 X：Y`
3. `X 正式发布 - Y`
4. `X 一战封神，我总结了 N 个 Y`
5. `从 0 到 1 带你速通 X`
6. `分享 N 个我最常用的 X`
7. `我花了 N，看到/做到了 X`
8. `X 悄悄上线了 Y，Z 真的来了`
9. `因为 X，整个 Y 都变成了 Z`
10. `X 背后，是一个被折叠的 Y`
11. `X 时代，为什么我劝你 Y`
12. `不要急着 X，先看完这篇 Y`
13. `X 不是 Y，而是 Z`
14. `一个极度实用的 X，帮你 Y`
15. `观察了 N 年，我把 X 分成了 N 个等级`
16. `X 的提示词技巧，就是没有技巧`
17. `AI 正在吞噬所有 X`
18. `这可能是最适合普通人的 X`
19. `所有用 X 的朋友，我都劝你先 Y`
20. `X 更新了，但我最想说的是 Y`

## 标题禁忌

- 不要只有“某某发布了”，没有判断。
- 不要只有“保姆教程”，没有对象和收益。
- 不要滥用“封神/最强/王者”，正文没有证据就别用。
- 不要为了像而加脏词，标题主要靠对象和判断，不靠粗口。
