# 原味语料分层

来源：沿用现有 `data/training-corpus-list.*`、`data/文章元数据总表.csv`、holdout 列表。本文件只补分层口径。

| 层 | 用途 | 原味关注 |
|---|---|---|
| 消费/品牌/公司事件 | 核心训练 | 主物件、供需、渠道、成本 |
| 风险科普/社会心理 | 辅助训练 | 普通人关系、风险边界 |
| 文娱/热点杂谈 | 语气训练 | 反差、短 punchline |
| 弱素材/不完整素材 | 反例 | 不硬写长文，先补事实 |
| holdout | 验证 | 只给题目/事实骨架，生成后对照 |

## 原味风险

- 只学短段和吐槽，容易变段子。
- 只学商业分析，容易变报告。
- 只学标题冲击，容易伤害事实边界。
