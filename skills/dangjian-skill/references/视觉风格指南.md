# 视觉风格指南

## 简报头

用户明确要“简报”时，可保留：

```text
练团子党支部
简 报
2026年第X期
练团子党支部 2026年X月
```

没有期数或月份时不要编，写 `第X期/2026年X月待核实`。

## 图片与说明

- 如果用户给图片，图片说明用括号短句：`（活动现场）`、`（党员代表分享学习心得）`。
- 不编图片内容；不知道画面就写 `（图片说明待补充）`。
- 领导调研图片说明只写确认的人物和位置；不确定不要写“左二/右三”。

## 排版节奏

- 标题可以双行：主标题偏气势，副标题偏说明。
- 简报正文不要过度分级，除非是发言稿或案例材料。
- 发言稿用“一、二、三”和“（一）（二）（三）”。
- 获奖宣传可以用连续长段承接荣誉背景、组织建设、品牌成果。
