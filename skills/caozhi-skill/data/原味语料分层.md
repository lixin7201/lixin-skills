# 原味语料分层

来源：沿用现有 `data/文章元数据总表.csv`、`data/文稿类型分布.md`、`data/小编语料分布.md`、holdout 记录。本文件只补原味分层口径。

## 分层

| 层 | 用途 | 原味关注 |
|---|---|---|
| 账号总风格 | 所有稿件底色 | 反差、短段、冷判断、事实边界 |
| 文稿类型 DNA | 主要写作路线 | 不同类型保留不同材料和节奏 |
| 小编/署名线 | 有样本时使用 | 不足样本不强行做个人 DNA |
| holdout | 验证 | 分类型、分小编单独看，不合并平均 |

## 原味风险

- 把多类型平均成一个声音。
- 把“槽值小妹”当真人小编复刻。
- 只学短段和热词，不学事实台账。
- 公共事件里补未核实细节。
