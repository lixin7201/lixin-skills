# 原味指纹

## 思考指纹

- 党建不是单独写理论，而是写“党组织如何进入业务一线”。
- 每个活动都要落到民生服务、组织凝聚、党员担当或地方发展。
- 组织建设常与业务数字化、平台资源、群众需求一起出现。
- 荣誉和领导调研都被写成“已有积累被认可 + 下一步继续做实”。

## 写作指纹

- 高频词：党建引领、初心使命、实干担当、凝心聚力、赋能发展、服务民生、同频共振。
- 高频结构：事实导语 -> 活动过程/工作做法 -> 党建意义 -> 下一步。
- 标题偏对仗和红色表达，如“初心/使命/担当/发展/民生/新篇章”。
- 常把业务分成就业、婚恋、消费、政务传播、公益服务等板块。

## 版式指纹

- 简报头真实存在，不要删除成普通公众号格式。
- 多数简报 500 字上下，段落短，但不是每句一段。
- 发言稿和品牌案例明显更长，允许密集长句和分级小标题。
- 结尾常有 `下一步`，这是原味，但不能空喊。

## 受保护的“土味/正味”

- 四字并列和口号感保留：它们是党建文体的一部分。
- `党建 + 业务 + 民生` 的重复强调保留，但要换事实支点。
- 正式称谓、组织全称、平台全称保留，不要为了顺口而乱简称。
- “荣誉既是肯定也是鞭策”这类表达可以用，但必须接具体下一步。

## 假像警报

- 只有红色词，没有具体活动和对象，不像。
- 只有营销夸赞，没有党组织动作，不像。
- 只有会议流程，没有党建意义和下一步，不像。
- 过度散文化、情绪化、网感化，不像。
- 把所有类型都写成同一个模板，不像。
