# Writing-DNA

使用顺序：先读账号总风格，再按素材选择文稿类型 DNA，最后叠加小编 DNA。

## 运行公式

`大宜宾 APP = 本地具体对象 + 小编出手方式 + 事实边界 + 评论区动作`

## 不同于公众号

- APP 稿更短、更直接、更依赖评论区。
- 不追求长篇完整叙事，追求用户能马上接话。
- 需要事实台账，尤其是网友曝光、招聘、突发、政务。

## 必读文件

1. `账号总风格.md`
2. `账号选题判断框架.md`
3. `文章结构模板.md`
4. 对应 `文稿类型/*DNA.md`
5. 对应 `小编风格/*-DNA.md`
6. `去AI味保真补丁.md`
