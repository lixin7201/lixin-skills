# 文章结构模板

## 通用党建简报

1. 简报头或标题。
2. 导语：时间 + 主体 + 活动/会议/调研 + 目的。
3. 过程：谁参加、在哪里、做了什么，按活动顺序写。
4. 党建意义：传承红色基因、凝聚思想共识、提升履职能力、服务群众。
5. 下一步：持续深化、常态化开展、立足岗位、服务地方发展。

## 长稿/发言稿

1. 主标题：对仗或品牌表达。
2. 副标题：说明党建与业务融合方向。
3. 背景：政策背景 + 企业/支部背景 + 痛点。
4. 主要做法：3-4 个小标题，最好都是“党建引领/机制创新/平台赋能/党群联动”。
5. 成效：数据、荣誉、群众服务、业务成果。
6. 启示/下一步：制度化、长效化、示范带动。

## 获奖宣传

1. 喜报/荣誉事实。
2. 荣誉含义：上级认可、权威印证、责任鞭策。
3. 长期积累：组织建设、党建品牌、业务融合、公益服务。
4. 代表性成果：就业、婚恋、消费、政务传播、公益等。
5. 下一步：珍惜荣誉、再接再厉、做优品牌。

## 领导宣传

1. 领导身份 + 时间 + 到访主题。
2. 接待和陪同人员。
3. 参观/汇报内容。
4. 领导反馈：只写已确认表态，不确认则写“认真听取、详细了解”。
5. 支部承接：以调研为契机，继续强化相关工作。

常规调研稿不要写成综合成绩汇报；围绕调研主题选材料，少量荣誉点到即止。

## 读书分享/研学会

1. 第几期活动 + 时间 + 主体。
2. 学习/分享主题。
3. 分享人和内容要点。
4. 与岗位实践连接。
5. 学习型党组织建设和下一步。

同类读书会稿件要短稳，避免每位分享人都套同一句“结合岗位实践”。代表分享足够清楚时，结尾不要再长篇拔高。除非用户明确要正式简报头，否则正文稿不加“练团子党支部/简报/第X期”抬头。

## 段落指标回归

- 简报类控制在 4-7 段；中位字数约 500-600。
- 发言稿可到 2500-4000 字；小标题层次清楚。
- 短段比例可高，但不要把同一个意思机械切碎。
- 结尾常见 `下一步/未来/将继续/将以`，但前文必须已经有具体事实。
