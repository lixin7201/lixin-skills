# 综合热点/观点拆解 DNA

- 训练样本：236 篇
- 定位：从一个热闹话题拆公共情绪，适合没有明确品牌/地域/娱乐单一标签但讨论度高的选题。
- 推进链：爆点 -> 第一反应 -> 反常识翻面 -> 多组材料 -> 时代情绪 -> 冷判断

## 标题

必须给具体对象和反差，不写“关于 X 的思考”。优先使用“第一批 X”“X 怎么就没人骂”“千万别 X”“这下，X 真尴尬了”等槽值式强入口。

## 开头

1. 先抛最新事实或最离谱的一幕。
2. 写读者第一反应。
3. 用一句话翻面：真正值得看的不是热闹本身，而是它暴露的逻辑。

## 正文材料

- 至少 3 组材料：平台讨论/网友现场、数据或历史变化、行业/人物/城市背景。
- 每组材料后必须给判断，不堆资料。
- 如果用户没有给事实，先列待核实，不编。

## 结尾

回收为一句冷判断或生活处境，不要鸡汤。

## 高分样例标题

- 刘若英《后来的我们》：我爱了很久，最后只能是朋友
- 《啥是佩奇》一夜爆红：别忘了那个偷偷爱你的人
- 千万别在深夜试探人性：这9.5分神综，刷新我三观
- 周杰伦被逼唱《学猫叫》：你看到了羞辱，我只觉得心酸
- 没人敢翻拍的《武林外传》，我13年后才看懂
