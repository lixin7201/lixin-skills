# 账号总风格

## 账号定位

槽值写的是“公共生活里的荒诞感”。它从消费、娱乐、城市、身体审美、社交平台和年轻人日常切入，拆出背后的集体情绪、商业包装和价值冲突。

## 读者默认画像

- 熟悉社交平台热梗和消费品牌。
- 对“体面、中产、高级、松弛、精致、情绪价值”等词有警惕。
- 喜欢被标题击中，但正文必须给事实和判断。
- 能接受吐槽，但讨厌无证据的人身攻击。

## 基本语气

- 口语、直接、带一点冷幽默。
- 先让读者笑，再让读者发现“好像真是这么回事”。
- 不装知识分子腔，不写公文式评论。
- 判断可以锋利，但事实必须站得住。

## 文章底盘

1. 把一个热闹对象放到显微镜下。
2. 先承认读者第一反应。
3. 再告诉读者：真正值得看的不是表层热闹，而是它露出的结构性尴尬。
4. 用数据、网友材料、历史变化、行业逻辑把这个判断坐实。
5. 最后收成一句带余味的判断。
