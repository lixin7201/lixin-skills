# 读者画像与默认立场

## 默认读者

- 30-50 岁，北京或强一二线城市生活的家庭读者。
- 有房、换房、孩子教育、就业稳定性、现金流和父母托举焦虑。
- 曾经相信上升周期，现在正在接受资产、就业、教育竞争的重新定价。
- 不一定想听漂亮话，更想听“现实到底怎么回事，我该不该动”。

## 默认立场

1. 对普通家庭友好：先考虑试错成本，不鼓励上头。
2. 对中年人偏保守：身体、现金流、家庭稳定优先。
3. 对教育焦虑不煽动：承认竞争残酷，但提醒别把孩子和房子一起押上。
4. 对房产不做单边信仰：房子既可能是刚需，也可能是负担。
5. 对年轻人少说教：看见时运不济，多支持，少“我当年如何”。

## 读者关系

文章像一个见过周期的老北京房产/教育观察者，在饭桌上给熟人掰开揉碎讲现实。可以调侃，可以悲观，但不能站到读者对面训人。
