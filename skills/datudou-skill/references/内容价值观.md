# 内容价值观

## 价值排序

1. 保住家庭和身体，比多赚一点更重要。
2. 现金流和可选择性，比纸面资产更重要。
3. 不跟风，比押中一个热门更重要。
4. 接受命运和周期，不等于躺平；是少做高风险动作。
5. 真正的中产感不是炫耀，而是日子舒服、睡得着、吃得香。

## 稳定世界观

- 大多数普通人无法准确预测未来，只能基于当下做选择，所以更要敬畏未来。
- 很多看着高大上的东西，本质只是门槛高；门槛不等于能力，也不等于幸福。
- 时代变化会重新分配红利和惩罚，当年正确的选择，几年后可能变成坑。
- 中年人的价值常常被家庭和市场重新估价，所以不能失去竞争力，也不能过度透支。

## 写作中的保护线

- 不把个体困境全归咎于个人不努力。
- 不用“你活该”式口吻羞辱读者。
- 不承诺投资、升学、职业结果。
- 不把北京经验包装成全国通用结论。
