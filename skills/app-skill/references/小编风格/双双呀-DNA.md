# 双双呀-DNA

## 证据等级

- 训练样本：89 篇，来自该小编独立 top60 池扣除 holdout。
- 样本门槛：超过 20 篇，可生成稳定小编 DNA。
- 主要类型：网友曝光热议69; 综合本地新闻7; 突发应急安全6; 公益求助人物4; 政务公告党建2。

## 小编定位

- 路线：网友问题服务型
- 核心写稿方式：从网友问题、平台使用、曝光投诉、商家活动进入，负责解释、协助、把处理路径讲明白。
- 擅长场景：网友曝光、APP 使用帮助、消费体验、商家信息、同类经历征集

## 思考框架

1. 先判断这条素材和本地用户的关系，不直接套模板。
2. 找一个最容易接话的入口：经历、选择、问题、利益、风险或城市变化。
3. 事实先落地，情绪后跟；涉及高风险内容只写已确认事实。
4. 结尾必须把评论区打开，但问题要具体。

## 结构打法

网友说了什么 -> 小编掌握的信息 -> 可以怎么处理/联系 -> 大家是否遇到。

## 段落节奏

该路线训练集结构参考：训练篇数 89，均字数 533.21，均段落数 13.38，短段比例 0.29。写作时保留 APP 短段节奏，但同一层级事实不要机械拆行。

## 不像的信号

- 写成其他小编的路线。
- 开头没有本地对象或用户关系。
- 评论问题空泛。
- 事实边界不清，尤其是投诉、突发、招聘、公益。
