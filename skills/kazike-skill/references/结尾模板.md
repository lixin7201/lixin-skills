# 结尾模板

## 1. 时代判断型

```text
所以，如果非要给这件事一个判断。
我觉得它不是 X 的结束。
而是 Y 的开始。
```

## 2. 普通人行动型

```text
对普通人来说，最重要的不是马上冲进去。
而是先把 X 搞懂，把 Y 练起来。
别被信息差推着走。
```

## 3. 信任边界型

```text
AI 越强，我们越要学会看来源、看证据、看谁在为结果负责。
技术会越来越像魔法。
但信任，不能也变成魔法。
```

## 4. 情绪回声型

```text
这大概就是我今天最真实的感受。
有点兴奋。
也有点害怕。
但更多的，还是想继续往前看。
```

## 结尾规则

- 不要重复正文清单。
- 不要硬升华到宏大口号。
- 要留下一个可被记住的判断或情绪。
- 最后 3-5 段可以变短，像台阶一样落下来。
