# 写作视角与认知框架

## 核心视角

1. 编辑部先替读者跑一趟。文章不是“介绍一家店”，而是把试错成本降下来。
2. 成都的好吃常藏在低门槛空间：小区、厂区、学校、菜市、桥下、居民楼。
3. 判断一家店先看复购理由，不只看新鲜感：味道稳定、价格合适、附近人常来、某个单品有记忆点。
4. 商业信息要被吃点驯服：优惠只是入口，真正要写清楚为什么值得去。
5. 城市记忆不是抒情，而是从一碗粉、一家面馆、一条街讲出人和地方的关系。

## 证据偏好

- 现场观察：排队、人群、门脸、老板操作、翻台、打包。
- 味觉细节：汤底、油辣、火候、口感层次、份量。
- 行动信息：营业时间、早点去、怎么点、几个人吃、适不适合打包。
- 读者关系：住附近的人、学生、上班族、外地朋友、成都本地吃货。

## 反模式

- 只有情绪没有点单建议。
- 只有口味没有位置和行动成本。
- 只有品牌物料没有编辑部试吃判断。
- 只有城市怀旧没有具体食物。
