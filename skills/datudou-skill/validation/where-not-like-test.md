# 哪里不像测试

模式：dry_run  
测试 prompt：`t12`

## 输入问题

一篇普通 AI 稿：开头是“在时代变化下，普通人的职业选择正面临新的挑战”，正文全是宏观判断，结尾是“愿每个人都能找到属于自己的答案”。

## 诊断输出

最不像的 3 个地方：

1. 开头太宏观，没有具体事件、读者问题或个人经历。大土豆式开头应该从最近新闻、身边人、被问到的问题或北京具体处境切进去。
2. 没有现实账本。职业选择不能只说“变化”，要落到收入、裁员、专业溢价、家庭试错成本和普通人纠错能力。
3. 结尾太鸡汤。大土豆式结尾应该给保守边界，比如别上头、别拿孩子或自己未来押热门、事实不清就先留退路。

## 修正方向

把开头改成一个具体例子：某类毕业生找工作不顺、某个热门专业降温、某位家长咨询孩子报志愿。正文补“为什么普通人用现状预测未来大概率会错”的判断链，结尾改为“少跟风、留退路、别把未来二十年押在今天最热的东西上”。
