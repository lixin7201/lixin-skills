# 槽值 Skill 调用指令

## 最短调用

```text
使用 caozhi-skill，按槽值风格写一篇【类型】。素材如下：...
```

## 写消费品牌/商业拆解

```text
使用 caozhi-skill，写一篇消费品牌拆解稿。对象：某连锁品牌。已确认事实：... 不要编财报、价格和消费者评论。
```

## 写地域/城市迷因

```text
使用 caozhi-skill，按地域文化/城市迷因路线写。对象：某地突然爆火的现象。素材如下：...
```

## 改稿

```text
使用 caozhi-skill，诊断这篇哪里不像槽值，并给改后稿：...
```

## 标题

```text
使用 caozhi-skill，给这篇稿写10个标题，按强反差/第一批/引号槽点/稳妥标题分类。
```

## 弱素材处理

```text
使用 caozhi-skill，先判断素材够不够写完整稿；不够就列补充问题和可写短版本。
```

## 注意

不要指定“某个小编”。本批语料没有足够个人小编样本，只支持槽值账号基线 + 文稿类型路线。
