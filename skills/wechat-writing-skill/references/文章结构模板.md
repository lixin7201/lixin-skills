# 文章结构模板

## 通用结构

```text
标题：对象 + 变化/冲突/收益
副标题：补一句具体影响或悬念
开头：3 句内交代对象和为什么值得点开
正文一：已确认事实
正文二：宜宾普通人的关系
正文三：图片/评论/现场线索承接
正文四：不确定边界或下一步
结尾：评论、转发、收藏、补线索
```

## 城市记忆结构

```text
再见/回来了/终于变了
→ 老宜宾人的旧印象
→ 现在看见的变化
→ 历史或生活记忆
→ 未来可能，但不写死
→ 你的记忆是什么
```

## 服务提醒结构

```text
谁需要看
→ 时间/地点/入口
→ 读者最关心的 3 个问题
→ 暂缺信息提醒
→ 收藏/转发给需要的人
```

## 人物故事结构

```text
人物最强结果
→ 为什么和宜宾有关
→ 关键经历/动作/物件
→ 处境和选择
→ 不消费苦难，落回尊重
→ 读者能怎么帮/怎么看
```

## 弱素材短观察结构

```text
一句编辑判断或泛生活画面
→ 已确认事实
→ 读者会关心的未确认问题
→ 征集线索/经历/图片
```
