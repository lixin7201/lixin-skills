# 原味语料分层

来源：沿用现有 451 篇语料、training/holdout 列表和既有评分。本文件只补分层口径。

| 层 | 用途 | 原味关注 |
|---|---|---|
| 人生观点/关系 | 核心训练 | 普通人处境、清醒判断 |
| 职场/金钱/成长 | 行动训练 | 留痕、边界、反馈、习惯 |
| 家庭/教育/亲密关系 | 关系训练 | 过度承担、默认、反噬、重新分工 |
| 敏感新闻/公共事件 | 边界训练 | 只借确认事实，不补前史 |
| holdout | 验证 | 只给题目/事实骨架，生成后对照 |

## 原味风险

- 只学数字分节，变成模板。
- 只学金句，变成鸡汤。
- 只学情绪共鸣，缺行动建议。
