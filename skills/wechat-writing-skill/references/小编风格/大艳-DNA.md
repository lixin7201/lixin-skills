# 大艳 DNA

## 证据边界

- 可用样本：71 篇。
- 硬证据：0 篇。
- 疑似主笔：69 篇。
- 来源需复核：0 篇。
- 排除/不建议个人蒸馏：2 篇。

这份 DNA 是“推断型编辑 DNA”。它适合用于稿件路线和结构选择，不适合用于个人语气复刻。任何署名、亲历、采访身份都必须回到素材事实。

## 适合题材

- 城建交通：拆迁、改造、路段、片区变化。
- 社会事件/维权：问题发生、影响范围、各方回应。
- 人物故事：把人物放回宜宾生活场景里。
- 老城、街区、公共空间变化。

## 写法核心

大艳线更像“问题推进 + 城市现场”：既写变化，也写这个变化对人有什么影响。它比纯记忆稿更强调动作、矛盾和现实结果。

常见推进：

```text
一个现场变化/问题
→ 影响到谁
→ 现在能确认什么
→ 历史或背景为什么重要
→ 普通人的感受与利益
→ 还需要等什么回应/结果
```

## 标题取向

- 用“已夷为平地/正式开始/封锁/开放/大变样”等动作词。
- 适合让地点、时间、变化结果同框。
- 维权类标题不先定性，不把冲突扩大成标签。

## 开头方式

- 先落现场：围挡、拆除、改造、路口、门店、学校、厂区。
- 再说影响：出行、生活、记忆、安全、便利。
- 背景后置，用来解释为什么这事值得写。

## 语言节奏

- 事实推进比情绪更靠前。
- 可以有怀旧，但怀旧要服务于“现在为什么重要”。
- 适合一段一问、一段一答的清晰结构。

## 可复用句法

- “这一次，变化已经不是传闻。”
- “对附近居民来说，影响最直接的是……”
- “从现场情况看，目前可以确认……”
- “至于后续怎么推进，还要等进一步消息。”

## 禁用

- 不能补官方回应。
- 不能补责任归属。
- 不能把普通人的猜测写成事实。
- 不能在没有素材时写成调查报道。
