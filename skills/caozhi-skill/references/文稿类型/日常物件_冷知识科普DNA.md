# 日常物件/冷知识科普 DNA

- 训练样本：33 篇
- 定位：食物、身体、生活物件、冷知识。
- 推进链：反常识问题 -> 生活经验 -> 科普/产业事实 -> 消费或身体处境 -> 轻吐槽结尾

## 标题

必须给具体对象和反差，不写“关于 X 的思考”。优先使用“第一批 X”“X 怎么就没人骂”“千万别 X”“这下，X 真尴尬了”等槽值式强入口。

## 开头

1. 先抛最新事实或最离谱的一幕。
2. 写读者第一反应。
3. 用一句话翻面：真正值得看的不是热闹本身，而是它暴露的逻辑。

## 正文材料

- 至少 3 组材料：平台讨论/网友现场、数据或历史变化、行业/人物/城市背景。
- 每组材料后必须给判断，不堆资料。
- 如果用户没有给事实，先列待核实，不编。

## 结尾

回收为一句冷判断或生活处境，不要鸡汤。

## 高分样例标题

- 街边两块钱一根的烤肠，里面到底是什么肉？
- 为什么还有人在淘宝买老人机
- 抖音里到底藏了多少宝藏男孩
- 为什么中国小孩那么爱上火？
- 影后周冬雨，为什么总演不好电视剧
