# De-AI Preservation Test

模式：dry_run  
测试 prompt：`t19`

## 输入问题

一篇已成稿有明显 AI 痕迹：包含“根据你提供的素材”“首先其次最后”“愿我们都能”，并且大量一句一段。

## 补丁结果

保留：

- 北京家庭账本
- 中年自保判断
- 口语化推进
- 事实边界

删除/替换：

- `根据你提供的素材` -> 直接进入事实叙述
- `首先/其次/最后` -> 按自然段推进
- `愿我们都能` -> 改成现实提醒或评论区落点
- 连续短句 -> 合并成 45-75 字左右自然段

## 回归结论

PASS。去 AI 后没有降低事实可靠性，没有把大土豆式克制悲观磨成通用温柔中文；段落节奏更接近训练集。
