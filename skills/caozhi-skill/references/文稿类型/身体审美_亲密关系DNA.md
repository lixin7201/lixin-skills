# 身体审美/亲密关系 DNA

- 训练样本：96 篇
- 定位：身体、审美、性别标签、亲密关系和家庭压力。
- 推进链：羞耻词/争议 -> 谁在制造标准 -> 普通人困境 -> 反规训判断 -> 边界提醒

## 标题

必须给具体对象和反差，不写“关于 X 的思考”。优先使用“第一批 X”“X 怎么就没人骂”“千万别 X”“这下，X 真尴尬了”等槽值式强入口。

## 开头

1. 先抛最新事实或最离谱的一幕。
2. 写读者第一反应。
3. 用一句话翻面：真正值得看的不是热闹本身，而是它暴露的逻辑。

## 正文材料

- 至少 3 组材料：平台讨论/网友现场、数据或历史变化、行业/人物/城市背景。
- 每组材料后必须给判断，不堆资料。
- 如果用户没有给事实，先列待核实，不编。

## 结尾

回收为一句冷判断或生活处境，不要鸡汤。

- 2.9分国产脑残剧收视第一，是对上亿女性的侮辱
- “老实人”徐峥六次恋爱失败：成年人爱情的残酷全在这了
- 那个凌晨三点不回家的女孩，一年没发过朋友圈了
- 那群在试衣间自嗨的女孩，你们够了
- 失去隐私的90后：每天，有10000人想拉黑父母
