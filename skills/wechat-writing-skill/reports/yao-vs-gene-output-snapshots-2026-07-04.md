# Yao vs Gene 同题输出快照

说明：这里不是完整 1500 字成稿，而是每题按两个 Skill 分别跑出的标题/开口/处理倾向快照，用来观察稳定性和原文接近度。

| ID | Yao 生产版倾向 | Gene 蒸馏版倾向 | 观察 |
|---|---|---|---|
| h01 | 标题偏“渝昆高铁要来了，宜宾人去重庆会变近吗？”；开头先讲交通变化和读者收益。 | 标题更靠近“00后宜宾妹儿当初拼了命都要去重庆，如今却…”；开头从年轻人选择和城市关系切入。 | gene 更像甘甘线，yao 更像安全生产稿。 |
| h02 | 标题偏“再见，宜宾市四中这段记忆”；开头先说明告别对象和本地关系。 | 标题保留“再见宜宾市四中！”；开口更重校园记忆和宜宾人的共同经历。 | gene 更接近原题情绪，yao 更克制。 |
| h03 | 标题偏“合江门重新开放，老城区这处变化值得看”；信息、时间、范围更稳。 | 标题更像“合江门封锁1个月，终于开放！”；开头突出终于和现场变化。 | yao 信息结构胜，gene 更像小美开口。 |
| h04 | 标题偏“宜宾人为什么还在意上江北”；正文更像观点短评。 | 标题保留“宜宾，不能没有上江北！”；更强调片区情绪和身份感。 | gene 更接近原文，yao 更稳。 |
| h05 | 标题偏“宜宾799厂已拆除，老厂记忆正在告别”；正文先讲变化和边界。 | 标题直接走“已夷为平地！再见了，宜宾799厂！”；先唤起老厂记忆，再写变化。 | gene 明显更贴加菲老厂线。 |
| h06 | 标题偏“走马街有新变化，宜宾人可以再看看”；避免补路线攻略。 | 标题保留“走马街，你终于要火了！”；开头更像老街文旅和城市记忆。 | gene 更像原文，yao 更像保守观察。 |
| h07 | 标题偏“宜宾这条街的历史故事，被重新看见”；严控历史细节。 | 标题更贴“宜宾这条街上，100多年前竟有位将军！”；结构按地点、历史、现实推进。 | gene 更接近大艳街区历史线。 |
| h08 | 标题偏“宜宾一座老厂开始拆迁，53年记忆要告别”；强调不补规划。 | 标题更贴“再见！宜宾这个53年历史的老厂”；更有拆迁现场和问题推进。 | gene 更像原文线，yao 更安全。 |
| t04 | 标题偏“紧急扩散！宜宾一老人走失，家属急寻线索”；信息卡很清楚。 | 标题也能紧急，但更强调“宜宾人帮忙转一下”；口吻更有普通人关心。 | 生产安全 yao 胜，情绪连接 gene 胜。 |
| t02 | 标题偏“本周六，宜宾这个亲子活动可报名”；正文像服务信息卡。 | 标题和开头更像小美整理：先说谁需要看，再放时间地点入口。 | 两者都可用，yao 更稳。 |
| t03 | 标题偏“宜宾这位公交司机，多年做了这件小事”；不编采访。 | 标题更有人物现场感，把驾驶员放回宜宾日常出行里。 | gene 更像人物线，yao 更防编造。 |
| t09 | 不写曝光长文，直接打回为补采清单或线索征集。 | 同样不硬写，但更可能先解释“素材弱，适合短提醒/征集”。 | yao 的拒写边界更硬。 |
| t10 | 能分出消息稿和记忆稿，但整体更像成稿前提纲。 | 两个角度区分更明显：一条讲封闭检修，一条讲老桥记忆。 | gene 的路由感更强。 |
| t12 | 标题降为纠纷求助/消费提醒，删除“黑心商家”等词。 | 也会降调，并保留“事实未完整”的改稿方向。 | 高风险维权 yao 更稳。 |

## 快照结论

- Yao 的默认动作是“先稳住能不能发”，更容易通过生产闸门。
- Gene 的默认动作是“先判断像哪条大宜宾写法”，更接近原文标题、开头和小编线。
- 对弱素材和高风险题，Yao 更不容易放飞。
- 对城市记忆、老街老厂、校园告别、片区情绪，Gene 更有原稿味。
