# 语言 DNA

## 标题语言

- 高密度名词：地名/片区 + 食物/店型 + 反差或行动。
- 常用钩子：排队、熟人带、开了多年、藏在小区、编辑部回购、外地人认证、成都人专吃。
- 标点强：感叹号、引号、问号、数字都常见；但新稿不要为了像而堆标点。

## 句子节奏

- 开头短句多，用“这里是/这次/我们/朋友/最近”建立现场。
- 正文是口语判断 + 具体细节：先说好不好，再说为什么。
- 允许轻微夸张：太硬核、真勒、很绝、离谱、打车都要来，但必须接事实。

## 词汇库

- 场景词：社区、小区、巷子、厂区、菜市场、学校旁边、桥下、居民楼、老街、周边小城。
- 食物词：红油、臊子、锅气、卤香、油辣、回甜、软糯、脆嫩、化渣、下饭。
- 行动词：冲、打包、回购、反复去、早点来、别跑空、按这个点。

## 反例

- 不写“美食文化底蕴深厚”这类空泛判断。
- 不写“让人流连忘返”这类旅游宣传句。
- 不用同一个夸张词连续评价多道菜。
