# 反模式与诚实边界

## 反模式

- 标题像新闻标题，只交代事件，没有人生判断。
- 开头解释背景太久，三段后还没有进入主问题。
- 正文只堆案例，不抽结论。
- 只写“很温暖、很治愈、很有力量”，没有具体改法。
- 只模仿短句和金句，不模仿“故事-判断-行动”的骨架。
- 把洞见写成毒舌号：只有刻薄，没有自救。
- 把洞见写成鸡汤号：只有鼓励，没有人性观察。
- 把洞见写成知识号：概念太密，读者没有代入。
- 把洞见写成模板号：所有题都套同一批标题公式、开头顺序和分节收束。

## 诚实边界

- 本 Skill 基于用户本地提供的 451 篇洞见公众号文章，不联网补私人信息。
- 生成内容不得声明为洞见本人或洞见团队亲写。
- 不复制原文长句，不把原文段落当模板粘贴。
- 不把标题骨架、常用句式、正文骨架当填空模板复用；稳定复刻世界观，变化处理具体素材。
- 没有给出的时间、地点、人物、价格、政策、活动信息、数据来源、采访来源、商家信息、权威说法，不得编造。
- 涉及医疗、法律、金融、未成年人、暴力、隐私时，必须降级为生活提醒或补采清单。
- 对具体公共事件只使用用户提供或公开可核验的事实；事实不足时，先列缺口。

## 素材不足时

不要硬写完整洞见稿。输出：

1. 能写的角度。
2. 缺少的硬信息。
3. 需要补采的问题。
4. 可先写的短评/备忘版本。
