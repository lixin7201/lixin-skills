# 开头模板

## 模板 1：最近事件入口

```text
这两天[事件]讨论挺多，我注意到一个细节：[细节]。
这事表面看是[表层]，但背后其实是[趋势/账本]。
```

适合高考、房产、裁员、政策、社会新闻。

## 模板 2：被问到入口

```text
最近有不少同学问我[问题]。
我想了想，这个问题不能简单回答，因为它背后至少有[数量]个变量。
```

适合买房、择校、职业、家庭账本。

## 模板 3：个人经历入口

```text
昨天/前几天我遇到一件小事，本来没当回事，后来越想越觉得挺有代表性。
```

适合中年、消费、家庭、年轻人。

## 模板 4：先声明边界

```text
先声明一句，本文不是广告，也不是建议大家立刻做什么。
我只是从我自己的经验出发，聊聊[对象]现在到底发生了什么变化。
```

适合房产、教育、风险话题。

## 模板 5：直给判断

```text
我现在越来越觉得，[判断]。
这个判断可能不讨喜，但对很多普通家庭来说，可能更接近现实。
```

适合反常识、命运感、中年自保。

## 开头禁忌

- 不要“在这个快速变化的时代”。
- 不要“今天我们来聊一个重要话题”。
- 不要先铺三段价值观。
- 不要把资料来源暴露成“根据你提供的素材”。
