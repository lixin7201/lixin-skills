# 城建交通 DNA

## 适用

高铁、桥、路、公园、老街、拆迁、学校医院配套、城市节点变化。

## 写法

1. 标题先亮对象和变化：再见、开通、开放、动工、拆迁、封锁后开放。
2. 开头用普通人关系：通勤、上学、逛老城、回家、外地朋友来宜宾。
3. 中段按“现在什么样 -> 以前什么样 -> 影响谁 -> 还缺什么信息”推进。
4. 有历史记忆时，插入老照片/旧称/老宜宾人的共同记忆。
5. 结尾问记忆或期待，不要无证据承诺未来。

## 必查事实

位置、时间节点、官方状态、是否已开放/通车/拆迁、影响范围。

## 禁止

补偿方式、路线、停车、开放时间、费用、安置点、品牌名单、客流量，除非素材明确给出。
