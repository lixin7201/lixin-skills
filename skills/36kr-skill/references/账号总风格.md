# 账号总风格

36氪的账号底色是“新经济商业报道”：先看到变化，再解释变化为什么有商业价值。它不靠情绪宣泄取胜，而靠信息增量、行业判断、资本/技术/用户三条线的组合。

## 选题气质

- 关注公司、产业、资本、技术、消费和城市产业的变化。
- 选题通常有一个明确新动作：融资、发布、上市、政策、财报、组织调整、用户行为变化或行业拐点。
- 文章必须回答“这件事为什么现在重要”，不能只是复述新闻。
- 趋势判断要带边界，少用绝对化结论。

## 共同写作动作

1. 先给事实锚点：谁、发生什么、金额/时间/产品/数据。
2. 再定义对象：这家公司/产品/事件到底属于什么赛道和问题。
3. 补行业背景：过去怎么运行，当前变量是什么。
4. 给商业判断：成本、效率、供需、竞争、政策、用户或资本逻辑。
5. 收束边界：机会在哪里，风险和不确定性在哪里。

不要把 36氪写成一个固定作者声线。调用时先走栏目/类型路线，再叠加作者线：`账号底色 -> 文稿类型 DNA -> 作者线增强`。
