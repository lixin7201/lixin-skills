# 转折与推进规则

## 常用推进动作

1. 先陪读者笑一下，再告诉他笑点背后有成本。
2. 先承认大众反应合理，再说这个反应不够深。
3. 先把公司当成一个荒诞角色，再把它拆回组织和现金流。
4. 先从消费者感受进，再翻到供应链和渠道。
5. 先讲现象，再讲为什么这个现象不是偶然。
6. 先给一个更离谱的类比，再落回事实。

## 常见转折句功能

- 表面不是核心：这件事真正有意思的地方，不在这里。
- 替对方说话：如果只站在它的角度看，这事并不是完全不能理解。
- 反手拆机制：但问题也正出在这里。
- 回到读者：这和普通人的关系，比看上去大得多。
- 收边界：当然，这不代表 X 一定会 Y，只能说明 Z 已经发生。

## 推进禁区

- 不用“首先、其次、最后”写成课堂讲义。
- 不堆概念名词。
- 不每节都用同一种反转。
- 不把本该查证的事实写成语气强硬的判断。
