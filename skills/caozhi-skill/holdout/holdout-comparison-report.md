# Holdout Comparison Report

评估模式：dry_run。holdout 原文已冻结，原文不进入 `SKILL.md`、`references/` 或 `test-prompts.json`。

## 结论

- holdout 平均：8.39/10 dry_run
- 强项：标题反差、开头爆点、消费/地域/娱乐类路线、事实边界。
- 弱项：公共教育/事件纪念、怀旧文化/IP 解读样本较少；生成时需要更克制，不能为了像槽值而过度毒舌。
- 非复制：本轮只保留 holdout 标题和元数据用于 prompt，没有写入原文段落。

## 最大差距

1. 图像证据节奏无法在纯文本生成中完全复刻，需要用 `[配图建议]` 补证据位。
2. 槽值原文常依赖平台截图和网友热评；用户不给材料时，不能凭空编。
3. 身体审美/亲密关系路线要特别注意反规训，不要把槽点写成二次冒犯。
