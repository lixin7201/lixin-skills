# 写稿流程操作手册

## 1. 素材门

先判断素材级别：

| 级别 | 标准 | 输出 |
|---|---|---|
| 强素材 | 有对象、时间、地点/城市、数据、用户观点、案例或经历 | 完整公众号稿 |
| 中素材 | 有主题和少量事实，但缺数据或案例 | 文章骨架 + 待补事实 + 可写短稿 |
| 弱素材 | 只有一句观点或标题 | 角度清单 + 5 个追问，不写成长文 |

## 2. 立判断

用一个句子先定调：

```text
这事表面是 X，本质是 Y。
对普通家庭来说，真正要看的不是 A，而是 B。
我不太建议大家 C，因为 D。
```

## 3. 拆账本

至少选一个账本维度：

- 钱：收入、房价、贷款、净资产、教育支出、消费。
- 分数：录取线、名额、区位、学校层级、竞争比例。
- 时间：通勤、陪伴、试错、身体恢复、学习周期。
- 风险：失业、流动性、政策变化、孩子承压、身体透支。

## 4. 排正文

默认顺序：

```text
标题
-> 事件/个人入口
-> 一句话判断
-> 分 3-5 层解释现实账
-> 插入个人经历或身边案例
-> 反常识转折：很多人误判在哪里
-> 给读者的保守边界
-> 克制结尾
```

## 5. 改稿动作

- 删掉空泛形容词，换成数字、位置、家庭角色或具体成本。
- 把“时代变化”落成“某类家庭会怎样”。
- 把连续短段合并成 50 字左右的自然段。
- 把鸡汤结尾改成现实提醒或评论区落点。
- 把“根据素材”改成直接叙述。

## 6. 自检

交付前逐项检查：

- 有具体入口，不是空降观点。
- 有账本，不只是情绪。
- 有北京/家庭/中年/教育/房产中的至少一个核心坐标。
- 没有冒充作者本人。
- 没有复制原文或 holdout 正文。
- 段落节奏接近训练集：中位约 52 字，短段不要泛滥。
