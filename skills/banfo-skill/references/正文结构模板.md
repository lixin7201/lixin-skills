# 正文结构模板

## 1. 快速热点拆解：4 节

1. 热点现场：读者都知道的表面信息，用更有戏剧性的方式重讲。
2. 第一反转：大家骂错/看浅/看偏了。
3. 机制拆解：谁的激励、成本、风险改变了。
4. 普通人关系：这事对读者意味着什么，最后给边界。

## 2. 消费体验文：4 节

1. 个人体验或细节震撼。
2. 这个细节为什么不是偶然。
3. 品类、供应链、场景、消费者心理拆解。
4. 回到买不买、值不值、怎么不被坑。

## 3. 商业公司文：5 节

1. 公司动作看起来离谱。
2. 先替它解释一层，避免单纯骂。
3. 拆老板意志、组织惯性、现金流或渠道压力。
4. 拆竞争格局和消费者关系。
5. 给判断：它真正危险/聪明/好笑的地方。

## 4. 风险黑产科普：8-12 节

1. 用一个与每个人相关的风险场景进入。
2. 解释黑话和链路。
3. 展示交易、获利、绕监管方式。
4. 讲普通人为什么会被影响。
5. 给自查和自保动作。
6. 结尾克制，不把恐惧卖成鸡汤。

## 5. 人物/商业史长文：8-13 节

1. 从一个低处、怪处或关键选择开场。
2. 按时间线讲命运，不急着评价。
3. 每个阶段给一个机制标签。
4. 把人物放回行业和时代。
5. 结尾不是封神，而是讲代价和边界。

## 段落节奏

- 训练集句子平均约 30 字，中位 25 字。
- 约四分之一句子不超过 15 字，用于梗、停顿、判断。
- 段落中位约 25 字，一段只承担一个动作。
- 短段服务梗、反转和停顿；同一组事实、吐槽、机制解释和方法提醒不要拆成连续微段。
- 长文二创/资料成稿时，铺垫句和答案要合并：不要把“问题来了”“这就很微妙”“这不是写错了”“当然能开”这类短句单独刷屏。
- 小标题多用加粗数字，少用正式报告标题。
