# 标题DNA

## 标题原则

大土豆标题通常直给对象和判断，不玩复杂修辞。标题要让读者知道：这篇和我的房子、孩子、钱、命、工作、生活方式有关。

## 可用标题公式

1. `简单说一下 + 现实对象`
2. `为什么 + 某类人 + 越来越难`
3. `中年人 + 一个直白判断`
4. `北京 + 具体账本/门槛`
5. `一个可能颠覆你认知的事实`
6. `看着高大上，无非只是 + 本质`
7. `别 + 常见错误动作`
8. `还是不能太 + 某种凑合/折腾`
9. `新的 + 趋势对象`
10. `再写一篇 + 育儿/教育/房产经验`
11. `我终于知道 + 某个现象原因`
12. `不能再 + 旧路径`
13. `你得 + 保守判断`
14. `最 + 某类现实状态的人`
15. `说说 + 近期事件和某类人`

## 标题避雷

- 不写“震惊全网、彻底变天、普通人必看”这类营销词。
- 不把标题做成公文或报告名。
- 不用过度精致的比喻，宁可朴素直白。
- 不承诺结果，如“这样买房稳赚”“这样培养孩子必上名校”。

## 标题输出格式

标题任务给 8-12 个，分为：

- 最像大土豆的直给标题
- 更适合传播的标题
- 更稳妥不冒进的标题

每组标注适用场景。
