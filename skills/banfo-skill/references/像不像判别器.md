# 像不像判别器

按 10 分制检查。

| 维度 | 10 分标准 | 不像时怎么改 |
|---|---|---|
| 标题 | 对象明确，强判断，能激发好奇 | 去掉报告腔，改成对象+反常识 |
| 开头 | 3 段内进入具体场景或反转 | 删宏观背景，从细节/提问/反常识进 |
| 机制 | 每个强判断都有成本、供需、渠道、激励解释 | 情绪词后补“为什么” |
| 普通人关系 | 能落到钱包、时间、面子、安全、工作 | 把抽象公司问题翻译成读者处境 |
| 语言节奏 | 短段落、梗和解释交替 | 拆长段，去掉公文句 |
| 素材纪律 | 不编事实，截图/数据有解释 | 标注未知，列待补材料 |
| 反转 | 至少有一次“表面 A，底层 B” | 增加视角翻面 |
| 结尾 | 收在边界、提醒或余味 | 删鸡汤升华 |
| 非模板化 | 不同题材结构不同 | 按文章类型换结构 |
| 非冒充 | 不声称原作者身份 | 改成风格参考写作 |
| 去 AI 腔 | 有具体物件承重，抽象判断不过量 | 用生活场景替换抽象总结 |

## 盲测高频输点

- 开头输：第一句没有把最难受的人或物件拎出来。改成“刚买车的人/押金单/合同/手机电量/退卡规则”先出现。
- 跨题输：把实习租房、教育、职场写成消费品牌稿。改成“规则成本 + 时间成本 + 普通人自保”。
- 去味输：只讲删词，没有给成稿。必须先给改后稿，原则放后面。
- 反模板输：三个开头只是换对象，骨架一样。必须换入口：人、物件、规则/账单。

## 诊断输出格式

```text
不像的地方：
1. 标题：...
2. 开头：...
3. 机制：...

修改原则：
- ...

修改稿：
...
```

## AI 感专项诊断

命中 3 条以上，先别继续写，必须重改：

- “能力、效率、交付、训练、系统、流程”这类抽象词密集出现。
- 连续出现“不是 A，而是 B”。
- 一段一个结论，但没有任何可见物件。
- 数字像论文摘要，没有变成读者能想象的场景。
- 每句话都很顺，没有转弯、停顿、反手拆解。
- 结尾像公号鸡汤或课程总结。

修改动作：

1. 给全文找一个主物件，例如作业本、考试卷、红笔、PPT、账单、货架。
2. 每一节至少放一个具体动作。
3. 把抽象句改成场景句。
4. 把“我认为/这说明”改成“你把 X 拿到 Y 里试一下”。
