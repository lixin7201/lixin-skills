# 开头模板

## 1. 深夜突发型

```text
昨天/今天凌晨，我本来已经准备睡了。
结果刷到 X。
看完第一反应是：这东西不太对劲。
不是普通更新，而是 Y 被改写了。
```

## 2. 亲测震惊型

```text
我这两天把 X 试了一遍。
说实话，试之前我以为它只是又一个 Y。
试完以后，我发现它真正离谱的地方不是 A。
而是 B。
```

## 3. 社群提问型

```text
最近后台/社群里，很多人都在问同一个问题：X 到底怎么用？
我本来没打算写。
但试完之后，我觉得有必要把这件事讲清楚。
```

## 4. 魔幻现实型

```text
这个世界，终于魔幻到我有点看不懂了。
事情是这样的。
我在 X 上看到了一个 Y。
它最离谱的地方，不是 Z，而是它说明了一个更大的问题。
```

## 5. 保姆教程型

```text
这篇不讲虚的。
如果你也想用 X 做 Y，照着下面的步骤来就行。
我会把能踩的坑、能省的时间、以及不值得折腾的地方，一次讲完。
```

## 开头禁忌

- 不要先铺 500 字行业背景。
- 不要第一段就堆术语。
- 不要没有素材就写“我试了”。
- 不要每篇都用同一个“故事是这样的”。
