# 怀旧文化/IP解读 DNA

- 训练样本：13 篇
- 定位：童年、老剧、国货、传统文化、IP 复活。
- 推进链：怀旧入口 -> 当年记忆 -> 今天重看发现的问题 -> 文化/时代解释 -> 余味判断

## 标题

必须给具体对象和反差，不写“关于 X 的思考”。优先使用“第一批 X”“X 怎么就没人骂”“千万别 X”“这下，X 真尴尬了”等槽值式强入口。

## 开头

1. 先抛最新事实或最离谱的一幕。
2. 写读者第一反应。
3. 用一句话翻面：真正值得看的不是热闹本身，而是它暴露的逻辑。

## 正文材料

- 至少 3 组材料：平台讨论/网友现场、数据或历史变化、行业/人物/城市背景。
- 每组材料后必须给判断，不堆资料。
- 如果用户没有给事实，先列待核实，不编。

## 结尾

回收为一句冷判断或生活处境，不要鸡汤。

## 高分样例标题

- “毁童年”的9.7分神作，藏着细思极恐的人生真相
- 被爸妈禁看的“脑残”神作，凭什么爆红20年
- 折磨上百万人的童年阴影，不该再出续集
- 你永远猜不透，滴滴司机能有多少秘密
- 即将消失的童年爆款，真的就那么不堪？
