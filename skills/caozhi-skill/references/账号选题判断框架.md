# 账号选题判断框架

## 优先选题

- 一个小对象能照出大情绪：西瓜、口红、羽绒服、预制菜、校服、鹅腿。
- 一个品牌/人物/地区正在被重新评价：从神话到失灵，从被嘲到翻身。
- 一个社交平台热词背后有真实压力：碳水脸、松弛感、妈生脸、精致穷。
- 一个看似离谱的现象其实有行业、历史或城市逻辑。

## 放大规则

|信号|可写性|
|---|---|
|有具体对象 + 公众讨论 + 反常识|强|
|只有观点，没有案例/数据/网友材料|弱|
|只有单一骂点，没有可拆结构|弱|
|能连接消费、城市、阶层、审美或代际情绪|强|
|需要严肃调查但事实不足|先列缺口，不写长稿|

## 选题动作

1. 找“槽点”：这件事为什么让人想笑、想骂、想转发。
2. 找“反转”：大家第一反应可能错在哪里。
3. 找“结构”：商业模式、审美规训、地域标签、平台算法、时代情绪。
4. 找“普通人处境”：它最终和读者的生活有什么关系。
