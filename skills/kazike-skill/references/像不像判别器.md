# 像不像判别器

## 评分维度

每项 0-10 分，总分低于 8 需要返修。

| 维度 | 看什么 |
|---|---|
| 标题 | 有明确对象、数字/亲历/强判断/冲突之一 |
| 开头 | 5 段内进入现场，不空铺背景 |
| 主判断 | 全篇只有一个核心判断 |
| 证据 | 强判断后有截图、结果、案例、路径或事实 |
| 语言 | 口语短段推进，不像新闻稿或 AI 说明书 |
| 结构 | 能从现象走到解释，再走到普通人关系 |
| 风险 | 酷点和坑点都写，不盲夸 |
| 结尾 | 有情绪回声或时代判断，不机械总结 |
| 诚实 | 不冒充作者，不编造亲测和内部消息 |

## 快速诊断句式

当用户问“哪里不像卡兹克”时，按这个格式输出：

```text
最不像的地方有 3 个：
1. 标题的问题是 X，应该改成 Y。
2. 开头的问题是 X，应该补 Z。
3. 正文的问题是 X，应该按“证据 -> 人话解释 -> 判断”重排。

然后给出修改稿。
```

## 一票否决

- 冒充作者本人。
- 编造亲测经历。
- 复制训练原文段落。
- 用空泛 AI 话术替代具体证据。
- 涉及高风险事实却不给边界。
