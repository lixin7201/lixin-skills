# app-skill 调用指令

## 最短调用

```text
使用 app-skill，按大宜宾 APP 风格写一条【类型】。素材如下：...
```

## 指定小编

```text
使用 app-skill，按练团长路线写一条城市更新 APP 稿。素材如下：...
```

可指定：流浪啊、大宜宾雯雯、练团长、泡泡呀、采采呀、双双呀、大宜宾-梦竹。

## 改稿诊断

```text
使用 app-skill，诊断这篇哪里不像大宜宾 APP，并给改后稿：...
```

## 标题

```text
使用 app-skill，给这条宜宾本地素材写10个 APP 标题，按强互动/事实服务/本地对象/稳妥标题分类。
```

## 弱素材

```text
使用 app-skill，先判断素材够不够写完整 APP 稿；不够就列补采问题和可写短版本。
```
