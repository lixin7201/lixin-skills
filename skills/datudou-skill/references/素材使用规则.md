# 素材使用规则

## 素材优先级

1. 用户提供的一手数据、截图、经历、楼盘/学校/政策材料。
2. 可核实的公开事实：成交、招生、政策、财报、人口、分数线。
3. 作者式个人经验推断：必须标注“一家之言、我觉得、在我看来”。
4. 情绪和金句：只能做辅助，不能当证据。

## 材料转写方式

- 数据不要堆，必须解释“对家庭意味着什么”。
- 案例不要煽情，必须指出它代表哪类普通人处境。
- 北京区位要具体：海淀、朝阳、东西城、丰台、五环、老破小、新房等。
- 教育材料要拆供需：孩子人数、学校名额、区内差异、家长成本。
- 房产材料要拆流动性：总价段、租售比、新房冲击、二手房卖方压力。

## 缺材料时

直接输出：

```text
这题现在不能直接写成大土豆式成稿，缺三类材料：
1. ...
2. ...
3. ...

如果先写，只能写成这个角度：
...
```

不要用“据说、有人说、网上都在传”硬撑事实。
