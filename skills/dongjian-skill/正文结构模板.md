# 正文结构骨架

以下是常见骨架，不是固定模板。先判断素材的核心痛点，再选择或改造骨架；不要让不同题材都长成同一篇文章。

## 骨架 1：三段式人生判断

适合：关系、内耗、金钱、认知。

```text
开头：故事或场景 -> 主判断
01：第一种表现/原因 -> 案例 -> 判断 -> 小行动
02：第二种表现/原因 -> 案例 -> 判断 -> 小行动
03：第三种表现/原因 -> 案例 -> 判断 -> 小行动
结尾：回扣标题 -> 3 句短收束 -> 推荐/行动
```

## 骨架 2：新闻/影视转洞见

适合：热点事件、热播剧、人物故事。

```text
开头：事件/作品引发情绪
01：复盘最有冲击力的细节
02：把细节转成普遍人性/关系模式
03：给普通人警醒或自救方法
结尾：不站队式拔高，落到边界和行动
```

## 骨架 3：职场单位三法则

适合：人在单位系列。

```text
标题：人在单位：A、B、C
开头：职场痛感或朋友案例
01：A，少做什么、多做什么
02：B，遇到什么情况怎么稳住
03：C，真正成熟的人如何处理
结尾：单位不是家，也不是战场；守住饭碗和心态
```

职场稿不能只写成情绪疏导。每篇至少落下 2-3 个具体动作：微信/邮件确认、节点反馈、责任边界、证据留存、延期说明、利益分配、必要时让上级做选择。判断要锋利，但落点要能帮普通人保住饭碗和体面。

## 骨架 4：清单行动稿

适合：50 条、7 件事、行动清单。

```text
开头：阶段变化/新年/低谷 -> 行动承诺
01-05 或 01-10：每节一个方向，每节 250-500 字
每节结构：问题场景 -> 一个案例/概念 -> 一组动作
结尾：不用宏大许愿，从一件小事开始
```

如果标题写了具体数量，正文必须交付对应数量。可以用分组小标题，也可以每条 1-3 句，但不能把“50 条”改成 20 条感悟；数量承诺本身就是读者点击后的信任契约。

## 骨架 5：关系避坑稿

适合：不要和某类人深交、关系结束、圈子。

```text
开头：一句关系判断
01：这类人的第一种消耗方式
02：这类人的第二种消耗方式
03：为什么你会被困住
结尾：真正的善良是有边界，离开也是自救
```

## 骨架 6：金钱/穷富认知稿

适合：穷相、存款、通勤、赚钱。

```text
开头：穷不是钱少，而是某种状态
01：习惯层面的穷
02：关系/情绪层面的穷
03：行动/心气层面的穷
04：可执行的守财或翻身动作
结尾：守住心气、秩序、骨气
```

## 骨架 7：家庭教育/亲密关系稿

适合：孩子、父母、婚姻、过度付出。

```text
开头：家庭故事/影视人物
01：关系失衡的表现
02：失衡背后的心理机制
03：普通家庭怎么止损
结尾：爱不是包办，付出不是勒索，边界才是长久
```

家庭关系稿要把“关系反噬链条”写完整：一个人承担太多，家人先习惯，再默认，最后把他的付出当成义务；付出者期待回报，期待落空后变成委屈，委屈再变成控制、指责或冷脸。洞见式落点不是单纯劝人别付出，而是劝人把爱放回边界、分工和尊重里。

过度付出题还要写“双边误读”：付出者说“我为这个家做了这么多”，家人听见的可能是讨债；家人说“你愿意做怎么又翻旧账”，付出者听见的是不被看见。这个错位比单纯委屈更像洞见的判断深度。

## 敏感新闻骨架修正

适合：暴力、刑事、未成年人、家庭冲突、法律风险事件。

```text
开头：只交代公开材料中已确认的一层事实
01：这件事最刺痛普通人的地方
02：把事件转成家庭边界/法律意识/养老秩序
03：普通家庭如何提前止损
结尾：不审判未知细节，回到规则、边界和底线
```

不要补写“第一次动手”“长期积怨”“邻居透露”等未提供细节，也不要用“第一次忍了、第二次忍了、第三次忍了”暗示重复前史。越是冲击性的事件，越要少复盘情节，多写底线判断。

## 反模板化改造

- 如果两个选题用同一骨架，至少变化开头入口、每节推进动作和结尾落点。
- 每节不要都写成同样的“案例 -> 判断 -> 行动”。可以改成：细节 -> 代价 -> 边界；场景 -> 反差 -> 选择；人物 -> 关系变化 -> 普通人动作。
- 判断句不能只换关键词。先写出素材独有的关系、物件、动作，再落回洞见式清醒判断。
