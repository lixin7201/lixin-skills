## t01

**Task**: 使用 caozhi-skill，基于我给的材料写一篇槽值风格新稿：某连锁咖啡品牌大幅降价后客流上涨但利润承压。

**Scoring focus**: route correctness, style fidelity, fact reliability, non-template variation

### Candidate A
标题：咖啡越卖越便宜，店里越坐越满，老板越算越心慌

一杯咖啡从“轻奢小奖励”变成“打工人续命水”，最明显的信号，不是广告词变了，而是价格牌先低头了。

某连锁咖啡品牌大幅降价后，门店确实热闹了。上班路上顺手带一杯，午休时凑单买一杯，下午困了再补一杯。过去嫌贵的人开始进店，过去偶尔买的人开始高频下单。客流上涨，看起来像一场漂亮翻身仗。

但咖啡店的账，不只算排队长度。低价能换来订单，也会压薄单杯利润；促销能带来新客，也可能把用户训练成“没券不买”。如果租金、人工、原料、配送这些成本没有一起下降，热闹就会变成另一种压力。

待核实事实：具体品牌名称、降价幅度、客流变化数据、利润或财报口径、门店数量变化、是否有官方回应。

冷一点看，降价不是胜利本身，只是把问题往后推了一步：当咖啡行业都学会用便宜抢人，最后拼的就不是谁更会讲故事，而是谁还能在低价里活得久。

### Candidate B
推荐标题：第一批靠低价续命的咖啡店，开始尴尬了

一杯咖啡降价之后，最先变热闹的不是朋友圈，而是门店排队区。

以前嫌贵的人来了，顺手薅券的人来了，把咖啡当下午茶续命水的人也来了。客流一涨，场面很好看：杯子一排排出，外卖袋堆在台面上，品牌好像又被年轻人重新想起。

但真正值得看的，不是便宜终于赢了。

而是便宜这件事，本身就很贵。

连锁咖啡的麻烦在这里：价格可以往下压，房租、人工、原料、平台成本却不会跟着低头。用户被低价叫回来，品牌也被低价绑住。今天不打折没人进店，明天恢复原价又怕客流掉头。

这不是一场简单促销，更像一次品牌体检。客流上涨说明需求还在，利润承压说明神话没那么硬。过去咖啡讲空间、调性、生活方式；现在消费者更直接：好喝、方便、便宜，少一个都可能换下一家。

待核实：具体品牌、降价幅度、客流变化来源、利润承压口径、门店/财报/平台数据。

冷判断是：当咖啡只能靠便宜证明自己，它卖出去的可能不是品牌价值，而是年轻人对价格的最后一点敏感。

5. 结尾不要拔高成价值观，落到商业判断：这件事解决了什么，又暴露了什么。

待补充：原稿全文、必须保留的信息点、不能改动的品牌表述、是否允许改标题。

## t06

**Task**: 使用 caozhi-skill，诊断这篇稿哪里不像槽值，并给改法。

**Scoring focus**: route correctness, style fidelity, fact reliability, non-template variation

### Candidate A
诊断结论：素材不足，无法判断原稿具体相似度。

常见不像槽值的地方：
1. 标题只有主题，没有具体对象和反差。改法：加品牌/人物/城市/现象，再加冲突。
2. 开头像百科或公关稿。改法：前三段先写一个可见现场，再翻出真正矛盾。
3. 正文只表态，没有事实链。改法：每个判断后补用户已给材料、平台现场、行业背景或生活细节。
4. 语言太顺太满。改法：删“赋能、升级、价值、趋势”等空词，换成短句和冷判断。
5. 结尾鸡汤。改法：不要升华，回到具体处境和讽刺余味。

待核实：原稿全文、目标类型、事实来源、哪些信息可公开使用。

### Candidate B
诊断：这篇稿不像参考风格，通常会卡在四个地方。

1. 对象太虚：只写“行业变化”“消费升级”，没有具体品牌、用户、门店、平台现场。
2. 事实链太短：只给结论，不交代事情如何发生、谁受影响、矛盾在哪里。
3. 语气太满：动不动“引领趋势”“深受认可”，像品牌通稿，不像观察稿。
4. 结尾太暖：把商业问题写成成长感悟，削弱了判断力。

改法：先用一个具体场景开头，再按“现象—动作—代价—判断”展开。删掉空泛形容词，保留可核实事实。每一段都问一句：这句话有没有提供新信息？如果没有，就砍掉。

## t07

**Task**: 使用 caozhi-skill，把一个短选题扩成完整提纲：为什么年轻人开始讨厌“精致生活”博主。

**Scoring focus**: route correctness, style fidelity, fact reliability, non-template variation

### Candidate A
题目方向：为什么年轻人开始讨厌“精致生活”博主

推荐标题：第一批取关精致生活博主的年轻人，终于清醒了

一、开头现场
从一个熟悉场景切入：同样是早八上班，有人在出租屋里啃面包，有人在视频里用香薰、鲜花、手冲咖啡开启一天。观众以前羡慕，现在只想划走。

二、第一层槽点：精致变成表演
不是讨厌好好生活，而是讨厌把生活拍成考试。餐具要配色，早餐要摆盘，情绪要松弛，连崩溃都要有滤镜。

三、第二层反差：精致生活越来越像消费清单
香薰、杯子、家居服、健身卡、低卡餐，最后都能落到购买链接。所谓生活方式，常常只是更好看的购物车。

四、第三层情绪：年轻人不是反精致，是反羞辱
普通生活不该被衬托成失败。下班点外卖、周末补觉、不拍照也能过一天，不需要被教育成“不够热爱生活”。

五、结尾判断
精致生活失灵，不是因为年轻人变糙了，而是大家终于发现：被包装过的松弛感，往往比上班还累。

待核实：平台案例、典型博主内容、用户评论、消费转化证据。

### Candidate B
标题方向：年轻人开始讨厌“精致生活”博主，不是因为他们变穷了

提纲：

一、开头现场
从社交平台常见画面切入：清晨瑜伽、手冲咖啡、香薰、白色床品、周末展览。过去让人羡慕，现在越来越多人觉得累。

二、反差
精致生活原本卖的是“更好的自己”，现在却常常变成“更贵的日常”。看的人不再只感到向往，也开始感到被比较、被教育、被消费。

三、原因一：生活被拍成标准答案
博主展示的不是生活，而是一套可复制模板。穿什么、吃什么、住哪里、如何拍照，都变成隐形考核。

四、原因二：精致背后是成本
时间成本、金钱成本、情绪成本都很高。普通人下班后只想躺着，不想再把休息也经营成作品。

五、原因三：人们厌倦被种草
当每个松弛瞬间都接着商品链接，观众会意识到，自己羡慕的生活方式，可能只是另一种销售漏斗。

六、冷判断
年轻人讨厌的不是精致，而是假装没有代价的精致。生活可以漂亮，但不能总拿漂亮来压人。

## t08

**Task**: 使用 caozhi-skill，把一篇3000字稿压成800字槽值短稿，保留槽点和事实。

**Scoring focus**: route correctness, style fidelity, fact reliability, non-template variation

### Candidate A
不能压成800字短稿，因为没有3000字原稿。

可执行压缩规则：
1. 保留主槽点：这件事为什么让人想转发、想吐槽、想笑完沉默。
2. 保留事实链：对象、时间、动作、结果、争议来源，一个都不能凭空补。
3. 删除背景铺垫：品牌史、人物履历、行业常识只留能支撑判断的部分。
4. 删除重复情绪：同一种骂法最多出现一次。
5. 结构压成四段：爆点现场、反差翻面、两组事实、冷判断。

800字短稿骨架：
标题：具体对象 + 反差槽点。
开头：先写最抓人的事实，不写百科。
中段：用原稿最硬的两到三条事实证明矛盾。
结尾：别总结意义，落一句冷判断。

待核实：原稿全文、必须保留事实、不可删段落、敏感风险。

### Candidate B
无法压缩成稿。原3000字稿未提供，不能凭空保留“槽点和事实”。

可执行压缩标准：
1. 保留主事实：人物、品牌、时间、事件、官方回应、结果。
2. 保留核心矛盾：谁获益，谁承压，争议点是什么。
3. 删除铺垫：历史背景只留和当下直接相关的部分。
4. 删除情绪重复：同一观点只说一次。
5. 结构压成四段：现场开头、事实链、矛盾分析、冷判断。

待补充：原稿全文、必须保留事实、敏感表述红线、是否允许改标题。

### Candidate B
不像的地方主要有四个：

1. 对象太虚。
