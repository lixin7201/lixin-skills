# 发言稿 DNA

## 适用场景

会议汇报、党建工作案例、书面发言、经验材料、品牌介绍、上级报送材料。

## 稳定结构

1. 主标题：红色表达或工作法名称。
2. 副标题：说明党建如何赋能业务或民生。
3. 背景介绍：企业/支部基本情况 + 政策背景 + 现实痛点。
4. 主要做法：3-4 个板块，常用“党建引领就业服务/婚恋服务/消费赋能/政务传播”。
5. 成效展示：数据、荣誉、服务人群、品牌建设。
6. 经验启示：制度化融合、党员一线、党群联动、行业带动。

## 写法要点

- 先解释为什么要这样做，再写做了什么。
- 每个小标题都要有业务对象，不要只有抽象党建词。
- 数据可以多，但必须来自素材；没有数据时写机制和动作。
- 允许长句和政策表达，篇幅可 2500-4000 字。

## 标题公式

- `以党建“红” + 绘就/赋能 + 发展“新”篇章`
- `创新 + 工作法/品牌 + 引领推动 + 高质量发展`
- `党建赋能 + 新兴领域/互联网企业 + 实干担当`
- `品牌名 + ：党建引领 + 数字经济/民生服务 + 新实践`

## 反例

- 把发言稿写成 500 字活动简报。
- 只写“我们高度重视党建”，不写机制、业务和成效。
- 每段都喊“党建引领”，但看不出就业、婚恋、消费、政务传播等落点。
