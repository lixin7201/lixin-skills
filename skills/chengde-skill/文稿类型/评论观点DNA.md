# 评论观点DNA

## 样本状态

- 样本数：17 篇
- 稳定度：可用
- 结论：可作为稳定路线使用

## 类型定位

用承德视角做城市判断：把一个事件写成承德的机会、危机、尊严或未来。

## 适用场景

城市排名、产业机会、公共争议、外地比较、体育文旅发展。

## 标题公式

问题/判断 + 承德命运感；常用“为什么、如何、答案、秘密”。

## 开头模板

用一个反常识判断或尖锐问题开局。

## 正文结构

现象 -> 反常识 -> 证据/比较 -> 本地逻辑 -> 判断 -> 行动或追问。

## 素材使用规则

数据、外地案例、历史节点、官方信息、城市对比。

## 转折方式

从情绪转到逻辑，再回到情绪。

## 结尾方式

给强判断，但保留事实边界。

## 事实核验点

数据来源、比较口径、时间窗口、引用来源。

## 不像该类型的反例

只有态度没有证据；套城市鸡汤；把愿望写成趋势。

## 训练样本标题示例

- 2023年，承德究竟是「几线城市」？
- 承德互联网，未来何处去？
- “房叔”的自白：我为什么劝你在承德尽快买新房！
- 老罗，也去滦平了！
- 你有没有发现，承德今年有点“反常”？
- 今天上午的磬锤峰，为啥这么热闹？
