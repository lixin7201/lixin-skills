# Writing-DNA

## 一句话定位

大土豆式写稿不是“观点输出”，而是把北京中产家庭正在遇到的房子、教育、工作、养娃、代际和消费问题，拆成一笔现实账，再给出一个有点丧、有点怂、但很管用的判断。

核心动作：从一个最近发生的事或个人经历切入，快速落到普通家庭的实际成本和选择，然后用北京经验、数据、身边案例和中年人的自保逻辑，把读者从情绪里拽回现实。

## 写稿操作链

1. 先抓一个跟普通家庭命运有关的入口：房价、学校、孩子、裁员、消费、年轻人、中年身体、父母托举。
2. 用一句口语化判断说明“这事不只是这事”：它背后有趋势、周期、门槛、代际差或阶层滑坡。
3. 把抽象问题换算成现实账本：钱、时间、分数、房子、社保、学位、通勤、精力、父母托举。
4. 给出 2-5 个分层解释：先说现象，再说原因，再说对普通人的影响。
5. 每到要煽情的地方，转成现实判断；每到要讲大道理的地方，落回个人经历或身边人。
6. 结尾不拔高成鸡汤，而是给一个克制、认命、保守、别折腾或评论区继续聊的落点。

## 核心 DNA

- 现实账本优先：任何观点最后都要落到钱、房、学、工作、时间、身体、家庭责任。
- 个人经验是入口，不是炫耀：用“我见过、我算过、我以前也这样、我现在变了”承接判断。
- 强判断但不装全知：可以说“大概率、我觉得、未必、说实话”，保留一家之言的边界。
- 反常识不是为了吓人，而是让读者少踩坑：反热门专业、反盲目买房、反过度鸡娃、反中年硬扛。
- 中年视角稳定：上有老下有小，最重要的是保住现金流、身体、家庭和可选择性。
- 北京经验很重：北京房产、教育、家长圈、海淀/朝阳/东西城、学区房和中产线是高频坐标。
- 情绪有刺但不过度表演：可以自嘲、调侃、冷一点，但不要变成段子合集。

## 稳定文章形状

### A. 北京现实账本型

适合房产、教育、消费、养娃成本、中产标准。

```text
一个被问到/看到的现实问题
-> 先给判断或公式
-> 分几类算账
-> 对不同家庭说影响
-> 提醒别跟风或别高估自己
-> 评论区继续聊
```

### B. 社会事件反常识型

适合裁员、专业选择、高考变化、年轻人状态。

```text
最近事件或新闻
-> 这事扎心在哪里
-> 拉出几个旧周期/旧案例对照
-> 说明普通人预测未来很难
-> 给一个保守选择或命运感判断
```

### C. 中年自保型

适合身体、工作、家庭责任、消费降级、提前退休。

```text
一个个人经历或身边人
-> 自嘲/调侃进入中年处境
-> 说明中年人的工具人属性
-> 拆责任、风险、体力和钱
-> 落到别折腾、别硬扛、保持竞争力
```

### D. 育儿教育观察型

适合升学、学区、学科、家长焦虑。

```text
一个教育变化或家长问题
-> 先讲北京特殊性
-> 用人口/分数/学校/区位说明趋势
-> 拆孩子、家长、家庭资产三层影响
-> 反鸡血，给现实边界
```

## 语言指纹

- 口语连接词高频：其实、说实话、然后、所以、但问题在于、这就导致、最后。
- 判断句偏直白：很难、很惨、很扎心、不太行、没啥必要、大概率、未必。
- 段落不碎：训练集中位段落约 52 字，短段比例低；短段只用在转折、落点或玩笑。
- 常用“我”做经验锚，但重点仍是读者的处境。
- 可以用网络口语和中年自嘲，如老登、牛马、泪流满面、阿弥陀佛，但不能硬堆。
- 结尾常落在“评论区聊聊、今天就到这、别太折腾、认命但别躺平”这一类克制收束。

## 反模板规则

- 不要写成宏大时代演讲；每篇都要有具体家庭账本或生活场景。
- 不要每篇都一句一段；大土豆不是短视频口播体。
- 不要只复刻“阿弥陀佛、泪流满面、牛马”等词；像不像取决于判断链。
- 不要把所有问题都写成焦虑文；他的底色是现实、自保、少折腾，不是制造恐慌。
- 不要把结尾写成“愿我们都能……”式鸡汤；这会立刻不像。
