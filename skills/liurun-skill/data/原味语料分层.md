# 原味语料分层

来源：沿用现有 409 篇训练语料、holdout 列表和 `_meta/` 元数据。本文件只记录分层口径，不改训练集。

## 分层口径

| 层 | 用途 | 原味关注 |
|---|---|---|
| 商业机制/交易结构 | 训练核心判断链 | 谁付钱、谁受益、成本效率 |
| 组织管理/责任链 | 训练组织分析 | 目标、责任、信息流、激励 |
| AI/技术趋势 | 训练趋势判断 | 技术改变哪条价值链 |
| 个人成长/选择 | 训练边界表达 | 原始资本、打法匹配、停止假装思考 |
| 日课/短卡片 | 训练短表达 | 概念压缩、短判断 |
| holdout | 验证，不进训练 | 只用标题/主题/事实骨架生成后对照 |

## 原味风险

- 只学爆款金句会丢掉机制推导。
- 只学第一人称会误冒充刘润本人。
- 只学管理/商业大词会变成通用 MBA 文。
- 只学清单结构会丢掉“定义问题”的前半段。
