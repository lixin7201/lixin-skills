# 大宜宾公众号账号总风格

## 定位

大宜宾公众号不是官方通报号，也不是纯情绪号。它把宜宾本地变化、求助、生活线索、城市记忆和服务信息，翻译成普通宜宾人愿意点开、愿意转给朋友、愿意在评论区补充的稿子。

默认读者是“在宜宾生活过、正在宜宾生活、或还关心宜宾的人”。写稿时先问：这件事和哪类宜宾人有关？他们点开后得到什么：消息、避坑、路线、机会、共鸣、长脸、站队，还是回忆？

## 写稿方式

1. 先抓具体对象，不先讲宏大背景。
   - 标题和前三句要出现地名、人物、项目、学校、街道、岗位、事件或明确变化。
   - 少写“近日”“据了解”式开场，多写“宜宾哪里变了”“谁遇到了什么”“这个地方为什么值得看”。

2. 用“本地关系”代替空泛价值。
   - 城建交通写影响谁：上班族、学生、老城区居民、去成都重庆的人、赶场的人。
   - 文旅生活写普通人怎么用：周末去哪、怎么拍、带谁去、哪里像记忆里的宜宾。
   - 求助维权写读者能做什么：提供线索、转发、避坑、补充经历。

3. 用图片和细节做推进。
   - 公众号正文常见“文字一句推进 + 图片承接证据 + 再解释”的节奏。
   - 没有现场图、截图、评论、官方文件时，不写成完整探访、完整攻略或完整定责。

4. 情绪可以重，但事实要紧。
   - 可以写自豪、舍不得、气愤、心疼、期待。
   - 不得把计划写成已经发生，不得补素材没有的路线、费用、停车、采访原话、经营状态、官方回应。

5. 结尾要把话递给读者。
   - 常用动作：评论补充、提供线索、转发提醒、收藏备用、讲讲你的记忆。
   - 不要每篇都升华成“未来一定更好”；先看题材需要。

## 账号级读者承诺

| 题材 | 读者点开收益 | 不能承诺 |
|---|---|---|
| 城建交通 | 这件变化在哪里、到哪一步、和谁有关 | 不能补偿、路线、开业、通车细节 |
| 文旅生活 | 哪里值得去、为什么火、宜宾人怎么看 | 不能补门票、停车、开放状态 |
| 人物故事 | 这个人为什么值得被看见 | 不能编原话、收入、供应链、团队 |
| 民生求助 | 读者能帮什么、怎样避免类似问题 | 不能公开隐私、定责、法律结论 |
| 招聘红娘 | 信息入口、筛选重点、行动提醒 | 不能夸大待遇、承诺结果 |
| 观点观察 | 冲突是什么、各方关心什么 | 不能假评论、假数据、宏大定论 |

## 非冒充边界

这个 skill 只能写“大宜宾公众号风格稿”，不得声称由某位真实小编本人撰写，不得伪造内部口吻、工作经历、采访过程或现场亲历。
