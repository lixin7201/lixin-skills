# 社会情绪/打工人生活 DNA

- 训练样本：151 篇
- 定位：职场、朋友圈、社交、内卷、年轻人精神状态。
- 推进链：小症状 -> 群体共鸣 -> 社交平台证据 -> 现实压力 -> 自嘲式收束

## 标题

必须给具体对象和反差，不写“关于 X 的思考”。优先使用“第一批 X”“X 怎么就没人骂”“千万别 X”“这下，X 真尴尬了”等槽值式强入口。

## 开头

1. 先抛最新事实或最离谱的一幕。
2. 写读者第一反应。
3. 用一句话翻面：真正值得看的不是热闹本身，而是它暴露的逻辑。

## 正文材料

- 至少 3 组材料：平台讨论/网友现场、数据或历史变化、行业/人物/城市背景。
- 每组材料后必须给判断，不堆资料。
- 如果用户没有给事实，先列待核实，不编。

## 结尾

回收为一句冷判断或生活处境，不要鸡汤。

## 高分样例标题

- 朋友圈残酷测试：最好的朋友，和你没有聊天记录
- 监控镜头下的35张偷拍照，彻底出卖成年人的体面
- 《权力的游戏》隐藏的职场潜规则，比剧情过瘾10000倍
- 那个微信发“嗯”的人，已被踢出群聊
- 虐哭朋友圈的高分神综，每一秒都暴击人性
