# 写稿判断框架

半佛式写稿不是先找金句，而是先找一个能被拆开的对象。对象可以是公司、消费品、热搜、电影、平台规则、社会情绪或一门看起来离普通人很远的生意。

## 核心判断链

1. 先找荒诞感：这个对象哪里反常、过头、拧巴、让人想笑或想骂。
2. 再找普通人关系：它为什么会影响钱包、时间、面子、尊严、安全感、工作或亲密关系。
3. 再拆机制：它背后是成本、供需、渠道、激励、品牌、老板意志、平台规则，还是信息不对称。
4. 再翻一次：表面看像道德问题，往往要拆成商业问题；表面看像商业问题，往往要落到普通人的处境。
5. 最后留边界：不知道的事实不补；涉及当前事件、公司数据、法律责任时，先核事实再判断。

## 文章不是观点喊麦

半佛式强判断要配机制解释。只说“离谱、疯了、傻眼”是不够的，后面必须回答：

- 谁从这件事里获益。
- 谁承担了成本。
- 谁被系统性忽视。
- 这事为什么不是偶然。
- 普通人能从中避什么坑，或看懂什么规则。

## 常见路线

| 路线 | 入口 | 正文承诺 |
|---|---|---|
| 商业公司拆解 | 公司动作、老板言论、财报、舆论风波 | 拆老板、组织、渠道、成本、用户心理 |
| 消费品牌观察 | 一次消费体验、一个产品细节、价格变化 | 拆品类、渠道、毛利、场景、消费心理 |
| 文娱产品拆解 | 电影、游戏、综艺、热点作品 | 从观感切入，拆叙事、产业、情绪 |
| 风险黑产科普 | 隐私、金融、骗局、法律边界 | 先讲可怕场景，再拆链路和自保动作 |
| 社会心理吐槽 | 亲密关系、父母、职场、学习 | 先给共鸣，再拆身份、面子和激励 |
| 人物/历史长文 | 人物命运、商业史、行业史 | 用时间线讲人，再从人拆时代机制 |

## 训练证据

本次训练集来自互动加权前 50%：608 篇训练、10 篇 holdout。高分样本标题包含：西贝的公关并不傻、《抓娃娃》简直该算恐怖片、全季酒店是怕我渴死吗？、蔚来的敌人只有傲慢的自己、腾讯会议，异地恋的核武器、《疯狂动物城》，其实是惊悚片、多少人把比亚迪当烘衣机用？、为了不让傻瓜电死自己，公牛插座承受了太多、水溶C100是怎么成为小红书神药的、罗森要毁了江浙沪便利店吗?、米村拌饭是预制菜的亲爷爷、《射雕：侠之大者》给我看笑疯了。
