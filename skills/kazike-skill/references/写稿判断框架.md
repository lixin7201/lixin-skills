# 写稿判断框架

## 先判断值不值得写

一个选题至少满足 2 条才进入成稿：

| 判断项 | 通过信号 | 不通过信号 |
|---|---|---|
| 新鲜度 | 刚发布、刚爆、刚被大量讨论 | 已经被写烂，没有新材料 |
| 可体验 | 能亲测、能截图、能复现、能给步骤 | 只能转述别人的结论 |
| 读者关系 | 普通 AI 用户、创作者、职场人、老板、学生能用上或避坑 | 只对极小圈层有意义 |
| 情绪强度 | 震惊、爽、怕、魔幻、破防、自豪、荒诞 | 只有温吞资讯 |
| 结构变化 | 改变工具链、内容生态、职业方式、信任关系 | 只是参数小升级 |
| 传播价值 | 读者愿意收藏、转发、评论、发给朋友 | 看完没有动作 |

## 卡兹克式角度选择

优先级从高到低：

1. 我亲自试了，发现它比宣传更夸张或更翻车。
2. 这个工具让普通人第一次能做过去只有专业团队能做的事。
3. 这件事表面是工具升级，背后是信任、创作、职业或商业规则变化。
4. 热点里有一个被大众忽略的信息差，我可以把它磨平。
5. 它很酷，但它也让某些人被折叠、被收割、被误导。
6. 这不是单点新闻，而是一个时代正在转向的证据。

## 事实门

- 产品、模型、政策、公司动作、价格、功能限制必须可核查。
- 没有亲测就不能写“我实测”“我试完”。
- 没有数据就不要写“最强”“第一”，除非明确是主观体验。
- 争议、封号、安全、违法、侵权、投资、医疗、教育等内容必须保留边界。

## 主判断落点

每篇只选一个主判断：

- 它让什么变简单了。
- 它让什么变危险了。
- 它暴露了什么信息差。
- 它改变了谁的工作方式。
- 它证明了哪个大趋势。
- 它让普通人该做出什么选择。
