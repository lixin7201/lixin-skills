# 地域文化/城市迷因 DNA

- 训练样本：133 篇
- 定位：省市标签、地方性格、文旅热梗、城市生活方式。
- 推进链：热梗/争议 -> 外地人误解 -> 本地逻辑 -> 历史/产业/生活解释 -> 重新理解

## 标题

必须给具体对象和反差，不写“关于 X 的思考”。优先使用“第一批 X”“X 怎么就没人骂”“千万别 X”“这下，X 真尴尬了”等槽值式强入口。

## 开头

1. 先抛最新事实或最离谱的一幕。
2. 写读者第一反应。
3. 用一句话翻面：真正值得看的不是热闹本身，而是它暴露的逻辑。

## 正文材料

- 至少 3 组材料：平台讨论/网友现场、数据或历史变化、行业/人物/城市背景。
- 每组材料后必须给判断，不堆资料。
- 如果用户没有给事实，先列待核实，不编。

## 结尾

回收为一句冷判断或生活处境，不要鸡汤。

## 高分样例标题

- 中国人羞于启齿几十年的事，被韩国片无情揭露
- 凭什么日本人全世界最瘦？
- 日本跟拍29年的神作，建议你谨慎观看
- 千万不要带广东人去四川
- 说 真 的，撸 鸭 好 爽
