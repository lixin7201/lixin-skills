# 账号选题判断框架

## 36氪式选题三问

1. 新变量是什么：融资、发布、上市、政策、价格、产能、组织、人群、技术或渠道。
2. 它改变了谁的成本/效率/分配：公司、用户、产业链、投资人、平台或监管。
3. 它是否能解释一个更大的趋势：赛道升温、泡沫退潮、商业化拐点、国产替代、全球化、AI 化、消费分层。

## 材料优先级

1. 最新事实、公告、财报、招股书、公司回应、监管或平台数据。
2. 产业链和客户证据：产能、订单、合作、价格、渠道、供应链。
3. 受访者/投资人观点，必须服务论证。
4. 用户场景和社交讨论，只做趋势信号，不能替代事实。

## 降级规则

- 没有核心事实：只写选题判断或采访提纲，不写完整报道。
- 没有数据来源：不写“市场规模”“增长率”“用户都在”。
- 没有采访材料：不写专访成稿，只写拟采访问答。
