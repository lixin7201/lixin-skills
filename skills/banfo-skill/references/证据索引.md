        - 训练集平均配图数：6.6。

        ## 选择口径

        `raw = 0.35*阅读 + 0.25*点赞 + 0.25*转发 + 0.10*喜欢 + 0.05*留言`，各项先 `log1p`。

        `weighted = 0.55*全局百分位 + 0.35*同年百分位 + 早期加成`。

        2018-2020 文章没有互动数据，不参与前 50% 排名，只作为历史风格背景。
