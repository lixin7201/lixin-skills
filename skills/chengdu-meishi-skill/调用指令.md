# 成都美食 Skill 调用指令

## 最短调用

```text
使用 chengdu-meishi-skill，按成都美食风格写一篇【类型】。素材如下：...
```

## 指定小编线

```text
使用 chengdu-meishi-skill，按 Mango 线写一个社区老店合集。不要编造地址和营业时间，缺失信息列待核实。
```

可指定：阿曾、初一、凯文、Mango、编辑部、桃子、莫莫、发吃报社。

## 改稿

```text
使用 chengdu-meishi-skill，诊断这篇哪里不像成都美食，并给改后稿：...
```

## 标题

```text
使用 chengdu-meishi-skill，给这篇成都肥肠粉稿写10个标题，按强钩子/街区单品/口语引号/稳妥商业分类。
```

## 商业稿

```text
使用 chengdu-meishi-skill，写成商家探店/活动稿。优惠规则以我给的为准，不要补时间、价格、地址。
```

## 弱素材处理

```text
使用 chengdu-meishi-skill，先判断素材够不够写完整稿；不够就列补充问题和可写短版本。
```
