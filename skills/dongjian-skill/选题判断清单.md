# 选题判断清单

## 可以写

- 能把读者的隐痛命名：内耗、穷相、过度付出、弱者叙事、无效沟通。
- 能形成强判断：不是 A，而是 B；越 A，越 B；真正的 X，是 Y。
- 能从个案推到普遍关系：家庭、单位、金钱、圈子、孩子、情绪。
- 能给读者一种“我知道该怎么做了”的出口。
- 有 2-4 个可讲案例，每个案例能服务同一个主判断。
- 有一句可收束的价值观：守边界、强心气、少内耗、先爱己、靠自己。

## 谨慎写

- 当前新闻：只借用已公开事实，不替代调查。
- 医疗、法律、政策、金钱建议：只写生活提醒，不写专业结论。
- 未成年人和家庭冲突：减少细节刺激，避免标签化。
- 商家、价格、活动：用户未提供硬信息时不编。

## 不该写

- 只有“发生了什么”，没有“这和读者有什么关系”。
- 只有流量争议，无法转化为人生判断。
- 只有金句，没有故事和材料。
- 只靠恐惧制造传播。
- 需要长调查才能成立的判断。

## 选题评分

- 读者痛点 0-3 分：是否击中普通人真实困境。
- 判断强度 0-3 分：是否能压缩成一句洞见式判断。
- 材料支撑 0-2 分：是否有足够故事/案例/概念。
- 风险可控 0-2 分：是否不需要编造硬事实。

低于 7 分，不写完整稿；改成短评、选题备忘或补采清单。
