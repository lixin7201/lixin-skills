# 消费品牌/商业拆解 DNA

- 训练样本：75 篇
- 定位：品牌、门店、价格、网红产品、餐饮连锁、消费神话失灵。
- 推进链：新闻/价格变化 -> 品牌神话 -> 用户态度变化 -> 商业逻辑 -> 为什么不灵了

## 标题

必须给具体对象和反差，不写“关于 X 的思考”。优先使用“第一批 X”“X 怎么就没人骂”“千万别 X”“这下，X 真尴尬了”等槽值式强入口。

## 开头

1. 先抛最新事实或最离谱的一幕。
2. 写读者第一反应。
3. 用一句话翻面：真正值得看的不是热闹本身，而是它暴露的逻辑。

## 正文材料

- 至少 3 组材料：平台讨论/网友现场、数据或历史变化、行业/人物/城市背景。
- 每组材料后必须给判断，不堆资料。
- 如果用户没有给事实，先列待核实，不编。

## 结尾

回收为一句冷判断或生活处境，不要鸡汤。

## 高分样例标题

- 网红民宿，一场刷屏朋友圈的骗局
- 初代智商税，把张雨绮骗惨了
- 网红女老师很多，我只服她一个
- ​一个有温度的汽车品牌，应该是什么样子？
- 1000块一瓶的天价水，真比农夫山泉好喝？
