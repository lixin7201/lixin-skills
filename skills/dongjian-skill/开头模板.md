# 开头模板

以下是进入路径，不是要照抄的文案模板。正式出稿时不要输出“开头”“模板”“示例”等元标签，也不要把这些路径连续机械套用。

开头优先从素材独有的痛点、动作或生活画面进入：饭桌、工位、聊天框、房间、转账、药盒、作息、衣服、会议记录。没有已确认硬事实时，可以写常见处境，但不要编造具体人物、时间、地点、原话。

## 1. 金句先行

适合：人生观点、关系、金钱认知。

模板：

```text
[一句短判断。]

前几天，看到/听到/重温了一个故事。

故事里的 [人物/场景]，表面看是 [现象]，其实暴露的是 [本文判断]。
```

示例：

```text
人这一生，最怕不是穷一阵子，而是先把心气过穷了。

前几天，看到一个卖旧物的帖子。

明明只是一次讨价还价，却把很多人的匮乏感照得清清楚楚。
```

## 2. 热点转人生

适合：社会新闻、影视剧、作文、热播剧。

模板：

```text
前两天，[事件/作品] 刷屏。

很多人只看到了 [表层情绪]。

但真正让我后背一凉的，是 [深层关系/人性/认知]。
```

## 3. 朋友/同事场景

适合：职场、人际、关系。

模板：

```text
这几天，朋友跟我讲了一件小事。

[场景细节]。

小事不大，却让我看见 [本文判断]。
```

## 4. 设问进入

适合：解释型标题。

模板：

```text
你有没有发现，越是 [现象] 的人，越容易 [结果]？

一开始，我也以为这是 [表面原因]。

后来才明白，真正的问题是 [深层原因]。
```

## 5. 反常识进入

适合：强判断。

模板：

```text
很多人以为，[常识判断]。

其实刚好相反。

真正决定 [结果] 的，从来不是 [A]，而是 [B]。
```

## 6. 书/名人观点进入

适合：认知、成长、金钱。

模板：

```text
[书/人物] 里有个说法，越想越扎心。

它说的不是 [表层主题]，而是 [读者处境]。

放到今天，尤其值得每个普通人警惕。
```

## 7. 生活画面进入

适合：低谷、家庭、身体、清单。

模板：

```text
一个人的状态，往往藏在最小的地方。

[房间/饭桌/工位/说话方式/消费习惯]，看似不起眼，却会慢慢塑造命运。
```

## 8. 清单承诺进入

适合：方法清单。

模板：

```text
新的一段时间，别急着给自己立宏大的目标。

先把 [身体/环境/钱/工作/关系/情绪] 这几件小事理顺。

下面这些行动，未必立刻改变命运，但能先把你从混乱里拉出来。
```

## 不像的开头

- “在当今社会，随着时代发展……”
- “本文将从三个方面分析……”
- 正文里出现“开头”两个字当段落标签。
- 一上来定义概念，像百科。
- 长篇介绍新闻背景，不进入读者痛点。
- 只说“今天分享一篇文章”，没有判断。
