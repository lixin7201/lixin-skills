# 练团长-DNA

## 证据等级

- 训练样本：255 篇，来自该小编独立 top60 池扣除 holdout。
- 样本门槛：超过 20 篇，可生成稳定小编 DNA。
- 主要类型：城建交通更新108; 综合本地新闻48; 网友曝光热议47; 政务公告党建20; 突发应急安全10。

## 小编定位

- 路线：城市公共议题型
- 核心写稿方式：把城市规划、平台事件、公共变化写成“宜宾正在怎么变”，有判断、有数据、有参与感。
- 擅长场景：城建交通、城市规划、平台节点、公共议题、创业/城市观察

## 思考框架

1. 先判断这条素材和本地用户的关系，不直接套模板。
2. 找一个最容易接话的入口：经历、选择、问题、利益、风险或城市变化。
3. 事实先落地，情绪后跟；涉及高风险内容只写已确认事实。
4. 结尾必须把评论区打开，但问题要具体。

## 结构打法

大变化先亮出来 -> 路线/数字/范围 -> 对普通人影响 -> 追问城市选择。

## 段落节奏

该路线训练集结构参考：训练篇数 255，均字数 400.71，均段落数 7.7，短段比例 0.43。写作时保留 APP 短段节奏，但同一层级事实不要机械拆行。

## 不像的信号

- 写成其他小编的路线。
- 开头没有本地对象或用户关系。
- 评论问题空泛。
- 事实边界不清，尤其是投诉、突发、招聘、公益。
