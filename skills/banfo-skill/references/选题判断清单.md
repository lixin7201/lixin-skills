# 选题判断清单

## 可以写

选题至少命中 3 条：

- 有一个具体对象：公司、产品、人物、平台规则、电影、消费场景，不能只有抽象概念。
- 有反常识：大家以为是 A，但更像 B。
- 有普通人关系：省钱、踩坑、工作、恋爱、父母、面子、安全、效率、尊严。
- 有机制可拆：价格、成本、渠道、供需、激励、信任、信息差、老板意志。
- 有素材支撑：用户给了事实、链接、截图、数据、经历，或者可先查证。
- 能用一个夸张但准确的比喻钉住对象。
- 写完读者能带走一个判断，不只是笑一下。

## 慎写

- 只有一句热搜，没有事实链。
- 只能骂，拆不出机制。
- 必须靠编造亲历、内部消息、价格、采访才能成立。
- 涉及法律、医疗、金融、事故、未成年人、公共安全，但素材不够。
- 只能复述别人的观点，没有自己的反转角度。

## 选题落点

半佛式落点通常不是“谁好谁坏”，而是：

- 这个系统如何把人推到某种选择里。
- 这个商品为什么能让人掏钱或骂人。
- 这家公司到底怕什么、赌什么、误判了什么。
- 这件事背后真正昂贵的东西是什么。
- 普通人怎么不被话术、情绪或复杂规则带着走。
