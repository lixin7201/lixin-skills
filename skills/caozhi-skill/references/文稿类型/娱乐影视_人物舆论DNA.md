# 娱乐影视/人物舆论 DNA

- 训练样本：86 篇
- 定位：明星、影视剧、综艺、公众人物争议。
- 推进链：热搜事件 -> 观众第一反应 -> 旧履历/作品证据 -> 人设翻面 -> 行业或审美判断

## 标题

必须给具体对象和反差，不写“关于 X 的思考”。优先使用“第一批 X”“X 怎么就没人骂”“千万别 X”“这下，X 真尴尬了”等槽值式强入口。

## 开头

1. 先抛最新事实或最离谱的一幕。
2. 写读者第一反应。
3. 用一句话翻面：真正值得看的不是热闹本身，而是它暴露的逻辑。

## 正文材料

- 至少 3 组材料：平台讨论/网友现场、数据或历史变化、行业/人物/城市背景。
- 每组材料后必须给判断，不堆资料。
- 如果用户没有给事实，先列待核实，不编。

## 结尾

回收为一句冷判断或生活处境，不要鸡汤。

## 高分样例标题

- “贾玲，你别装了”
- 女明星一句话惹哭何炅：人生最残忍的，是来不及告别
- 《哪吒》被忽视的4个隐藏剧情，够硬核
- 被误解25年的高分神剧，一半成年人被它治愈
- 韩寒怎么走到了这一步
