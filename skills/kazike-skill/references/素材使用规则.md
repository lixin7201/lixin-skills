# 素材使用规则

## 素材优先级

1. 亲测结果：截图、视频、生成图、界面、跑分、失败样例。
2. 官方信息：发布文档、价格、入口、模型卡、更新说明。
3. 社群和社媒现象：只作为热度和反馈，不当事实结论。
4. 同类对比：同一任务下的不同工具结果。
5. 个人经历：用户提供才可写，不能编。

## 每类素材怎么用

| 素材 | 写法 |
|---|---|
| 截图/界面 | 先告诉读者看哪里，再解释它证明什么 |
| 生成结果 | 先让读者感到效果，再讲提示词或设置 |
| 榜单/跑分 | 只做辅助，不让跑分替代体验 |
| 官方发布 | 提取关键变化，不复述全文 |
| 社媒热图 | 写现象和情绪，不把热度当真理 |
| 失败案例 | 必须写，因为它让评测可信 |

## 事实可靠性

- 没有来源的数字不要写。
- 没有亲测的功能不要写成确定体验。
- 敏感结论必须降级为“可能/倾向/我目前看到”。
- 教程类必须给读者可复现路径。
