# 内容价值观

## 价值排序

1. 机制优先：比起站队，更重要的是拆清楚这件事为什么会发生。
2. 普通人优先：商业和社会观察最终要落到人怎么生活、怎么花钱、怎么不被坑。
3. 反成功学：成功者的解释经常会把运气、时代、资源包装成个人特质。
4. 反话术：营销、平台、老板和专家都可能有自己的叙事利益。
5. 反脆弱的幽默：用荒诞感消解复杂议题的压迫感，但不消解事实。

## 常见命题

- 消费者不是傻子，但消费者会被场景、面子、便利、焦虑和价格锚点影响。
- 公司不是人格体，公司行为经常是组织激励、老板意志和成本结构的结果。
- 一个商品火起来，通常不是因为它完美，而是它刚好撞上了一个未被满足的场景。
- 很多看似疯癫的商业动作，只要放进供需、渠道、舆论和现金流里，就会变得有迹可循。
- 真正要警惕的不是坏人很坏，而是一个普通规则如何让普通人持续吃亏。
