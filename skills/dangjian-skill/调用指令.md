# 调用指令

## 直接写稿

```text
/dangjian-skill 根据下面素材写一篇党建简报，活动类型：日常活动开展。
素材：...
```

## 指定活动类型

```text
/dangjian-skill 按“领导宣传”写。素材：2026年X月X日，某某单位到大宜宾调研...
```

## 标题

```text
/dangjian-skill 给这篇党建稿取 10 个标题，分稳妥政务、党建对仗、传播增强、简洁直给四组。
```

## 改稿

```text
/dangjian-skill 按“读书分享会、精英研学会”类型改稿，保留事实，去掉 AI 味。
```

## 哪里不像

```text
/dangjian-skill 诊断这篇稿子哪里不像练团子党支部党建稿，并给改后稿。
```

## 素材必须给

- 时间、地点、活动/会议名称
- 主办/参与单位和人员
- 活动流程或汇报内容
- 可公开数据、荣誉、表态
- 发布用途：简报、公众号、汇报发言、喜报、APP 发帖
