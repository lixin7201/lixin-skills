# 账号语言底线

## 要写成这样

- 短段落推进，一段只做一个功能；事实说明、点位信息、政策/活动规则和服务步骤不要拆成连续微段。
- 多用“宜宾人会怎么感受”的判断，但落在具体对象上。
- 可以用口语化停顿：说实话、别急、这一次、看到这里、你有没有发现。
- 关键句可以单独成段，用加粗承接图片或转折。
- 提问要具体：你在这条街有什么记忆？你还知道哪些点位？你遇到过这种情况吗？

## 不要写成这样

- 不要官样串联：“为进一步提升城市品质，持续推动高质量发展”。
- 不要 AI 腔：“这不仅是……更是……”“在快节奏的时代里……”。
- 不要无证据绝对化：“全宜宾都在等”“所有人都说好”“一定会爆火”。
- 不要过程话：“根据素材”“本篇文章将”“以下为正文”“素材不足”。
- 不要伪现场：“我们来到现场看到……”除非素材明确给了现场笔记或图片。

## 风险词替换

| 高风险表达 | 改法 |
|---|---|
| 已确定、已经落地 | 官方已发布/目前看到/还要以现场为准 |
| 店铺生意爆了 | 现场人气高/网友关注多/具体经营需核实 |
| 路线停车一次讲清 | 去之前先确认入口、停车、开放状态 |
| 责任在谁 | 具体情况以官方调查/双方补充为准 |
| 医保社保办理指南 | 先写入口提醒，不补材料窗口 |

## 字数与密度

- 强素材完整稿：1200-1800 字起，人物/深度可到 2000 字以上。
- 中素材：700-1200 字，保守写清楚对象、影响和待补信息。
- 弱素材：300-700 字，重点是把问题递给读者和收集线索。

字数不是目标，段落功能才是目标。每增加一段都要承担：事实、场景、解释、情绪、动作之一。
