# 原味语料分层

生成日期：2026-07-09

## 结论

本技能包不能只学习高互动文章。大土豆的原味来自“爆款判断 + 日常随笔 + 普通家庭账本 + 个人边界”共同形成的稳定写法。训练时保留普通样本和弱样本，避免把技能推成只会写爆款焦虑文。

## 结构基线

| 指标 | 全量均值 | 训练集约束 |
|---|---:|---|
| 中文字符数 | 989 | 常见成稿约 900-1100 字 |
| 段落中位长度 | 57.6 | 正常段落约 45-75 字 |
| 每千字段落数 | 17.0 | 不写成短句瀑布流 |
| <=20 字短段比例 | 0.07 | 短段只给转折、玩笑、落点 |

## 使用规则

- 高互动样本给“选题和判断强度”，普通样本给“日常节奏和真实边界”。
- 房产/教育稿可强判断，但必须有事实边界。
- 人情/家庭稿最容易被写成鸡汤，必须补家庭角色、钱、身体、时间或责任账本。
- 个人经历类稿不能伪造作者亲历；用户不给真实经历时，只能写观察。
- 不把“老登、牛马、阿弥陀佛”等口头词当作核心 DNA；这些只是少量情绪峰值词。
