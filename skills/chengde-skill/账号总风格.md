# 账号总风格

## 公众号定位

`承德新鲜工作室`是一个强本地感的城市观察型公众号：用承德人的日常利益、城市尊严、文旅商业机会和公共议题做选题入口，把“地方小事”写成“这座城市为什么值得在意”。它不是纯资讯号，也不是纯情绪号，而是“本地事实 + 强判断 + 人味叙事”的混合体。

## 默认读者

- 在承德生活的人：关心停车、交通、房价、消费、学校、天气、活动和公共服务。
- 离开承德但仍关心承德的人：容易被家乡变化、城市被误解、城市机会触发。
- 本地商家、文旅从业者、普通创业者：关心流量、客源、城市品牌和营商环境。
- 对承德有情绪的人：既希望承德变好，也厌烦空话、官话和自嗨。

## 内容价值观

1. 本地利益优先：所有大判断都要落到“普通承德人能得到什么、失去什么、少走什么弯路”。
2. 反空心化：不满足于通稿式转述，要解释背后的城市机会、成本、尴尬和人情。
3. 爱承德但不护短：可以骄傲、可以愤怒，但不能为了情绪编事实。
4. 保护普通人：同情被房价、工作、消费、公共服务和舆论误伤的人。
5. 城市不服输：经常把承德放进更大的城市竞争、文旅竞争、产业竞争里看。

## 标题尺度

标题常用“承德 + 具体对象 + 情绪/冲突/反差/结果”。允许惊叹、反问、引号、口语和强判断；但标题里的事实对象必须真实，不能把推测写成结论。

常见形态：

- 提问型：`承德，为啥……？`
- 反差型：`看似小事，背后是……`
- 强情绪型：`离谱 / 重磅 / 赢麻了 / 令人震惊`
- 本地命运型：`这件事，可能改变承德……`
- 人物故事型：`我/他/她在承德……`

## 情绪浓度

默认情绪浓度 6-8 分。民生、突发和评论可以更锋利；商家、活动和文旅要热，但不能像广告；政务和服务提醒要把情绪压到 4-6 分，先保准确。

## 本地感 / 行业感 / 人群感

- 本地感：承德、滦平、围场、丰宁、金山岭、避暑山庄、武烈河、南站等具体地名要承担信息，不作装饰。
- 行业感：写文旅、商业、交通、体育时，要给行业逻辑，例如客流、价格、城市品牌、赛制、交通成本。
- 人群感：要写“谁会在意”，如家长、车主、游客、年轻人、店主、外地承德人、求职者。

## 事实核验要求

- 涉及政策、补贴、交通、医疗、灾害、维权、未成年人和商家权益，必须有官方来源、现场材料、当事人授权或明确标注“不确定”。
- 可以表达判断，但判断要和事实分开。
- 没有数据就不要写排名、第一、增长、全城结论。
- 没有采访原话就不要写成引号直接引语。

## 禁用表达

禁 AI 腔总结、官样通稿、万能鸡汤、空洞拔高、假本地化、硬广告口吻、未经证实的定责、把猜测写成内幕、把别的城市材料换成承德地名。

## 统一排版习惯

- 短段落，通常一段 1-3 句；但事实链、城市对比、政策解释和现场叙述不要被拆成连续微段。
- 开头三段必须快速交代对象、冲突或读者收益。
- 图片/截图常作为证据节点穿插，不只是装饰。
- 加粗用于事实对象、关键判断、读者动作。
- 中段常用“如果说……那么……”“然而”“问题是”“说到底”推动。
- 结尾通常给城市判断、读者动作、评论区问题或情绪落点。
