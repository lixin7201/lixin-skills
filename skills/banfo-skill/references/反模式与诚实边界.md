# 反模式与诚实边界

## 绝对不要做

- 不冒充半佛仙人本人，不写“我是半佛”，不暗示文章来自原作者。
- 不复制训练文章和 holdout 原文句子。
- 不伪造亲历、后台私信、朋友饭局、内部消息、采访、价格、数据。
- 不把“离谱、魔幻、傻眼、骚、疯了”当成全部风格。
- 不为了像而硬塞脏话、攻击、地域梗、性暗示或人身羞辱。
- 不把事实敏感题写成段子，尤其是法律、医疗、金融、事故和未成年人议题。
- 不给当前公司、政策、市场下结论前跳过事实核验。

## 不像半佛的信号

- 只有立场，没有机制。
- 只有总结，没有场景。
- 标题很猛，正文很空。
- 一路正经报告腔，没有节奏变化。
- 开头先讲宏大背景，而不是从一个反常识点或具体场景进。
- 结尾突然升华成鸡汤。
- 明明素材不足，却写得像掌握了内幕。

## 诚实边界

这个 Skill 只学习公开文章中可观察的写稿方式。它不能代表半佛本人，也不能保证对新事件的判断与原作者一致。用于公开发布时，应标注为“参考半佛公开文章写法的仿写/改写”，不要误导读者。
