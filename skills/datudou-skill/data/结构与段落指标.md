# 结构与段落指标

统计口径：训练集 40 篇；去掉 frontmatter、标题、作者行、原文链接、互动数据；图片单独计数。

| 指标 | 训练集结果 |

| 平均中文字符数 | 981 |
| 中位中文字符数 | 948 |

| 段落中位长度 | 52.0 |

| 连续 <=20 字短段最大 run 均值 | 0.68 |

## 可执行约束

- 正文不是一句一段的短视频脚本，正常段落以 45-75 个中文字符为常态。
- 短段只用于开场钩子、判断落点、转折、玩笑或评论区召唤；连续短段超过 3 段要检查是否机械碎片化。
- 常见结构是「事件/私事入口 -> 个人判断 -> 现实账本/数据/案例 -> 反常识拐弯 -> 命运感或评论区落点」。
- 图片通常是证据或截图，不是装饰；没有用户提供图片时，不要编造截图。
